== NAME
ps - Alternate command name for *postscript*.

== SYNOPSIS

[source]
----
ps [-a author] [-b] [-c r/g/b] [-f font] [-s size] [-t title] [-x offset] [-y offset] file
----


[[description]]
== DESCRIPTION

The *ps* command shares its core implementation with *postscript*.
It writes the current display to a PostScript output file.

The *-c* border color accepts 8-bit RGB channels in _r/g/b_, _r,g,b_, or
_r;g;b_ form, or as three separate arguments (_r g b_).

See *postscript* for the detailed behavior and option meanings.


[[examples]]
== EXAMPLES

.Write the current display to a PostScript file using the short command name.
[example]
====
mged> **ps -t "Wheel View" wheel.ps**

Performs the same operation as *postscript* and writes the current display to
_wheel.ps_.

====
