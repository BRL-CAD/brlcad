== NAME

get_bot_edges - list the edges of a BOT primitive

== SYNOPSIS

[source]
----
get_bot_edges [--help] bot
----

== DESCRIPTION

The *get_bot_edges* command lists the unique edges of a BOT (Bag-of-Triangles)
primitive. Each edge is written as a pair of vertex indices.

=== Options

*-h*, *--help*::
Print command help without requiring an open database.
