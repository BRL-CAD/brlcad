== NAME
knob - Provides a method for simulating knob input. With no options, it will display the current values for the knobs.

== SYNOPSIS

[source]
----
knob [--help] [--incr] [--view-coords | --model-coords] [--origin v|m|e|k] [action [value] ...]
----


[[description]]
== DESCRIPTION

*The *knob* command is used internally by MGED in the processing of knob input devices and is not recommended for users.* The *knob* command provides a method for simulating knob input. With no options, it displays the current knob values. It accepts a sequence of actions: *zap*, *zero*, *clear*, *stop*, and *calibrate* take no value; every knob identifier takes one finite numeric value. *zap*, *zero*, *clear*, and *stop* reset the rate-based values. *calibrate* resets the absolute translation baselines.

If the _-i_ / _--incr_ option is specified, each numeric value is applied as an increment. The _-v_ / _--view-coords_ and _-m_ / _--model-coords_ options select the coordinate system and cannot be combined; if neither is given, the current view coordinate setting is used. The _-o_ / _--origin_ option selects the rotation origin: _v_ (view, the default), _m_ (model), _e_ (eye), or _k_ (the current keypoint). Rotation values are in degrees and translation values are in local units. The available knob identifiers are:


* *x* -- rate-based rotation about horizontal axis.
* *y* -- rate-based rotation about vertical axis.
* *z* -- rate-based rotation about axis normal to screen.
* *X* -- rate-based translation in horizontal direction.
* *Y* -- rate-based translation in vertical direction.
* *Z* -- rate-based translation in direction normal to screen.
* *S* -- rate-based Scale or Zoom.
* *ax* -- absolute rotation about horizontal axis.
* *ay* -- absolute rotation about vertical axis.
* *az* -- absolute rotation about axis normal to screen.
* *aX* -- absolute translation in horizontal direction.
* *aY* -- absolute translation in vertical direction.
* *aZ* -- absolute translation in direction normal to screen.
* *aS* -- absolute Scale or Zoom.

The command rejects unknown identifiers, missing values, nonnumeric values, and a mixture of view and model coordinate options before modifying the view.


[[examples]]
== EXAMPLES

The example shows the use of the *knob* command with the _y_ option to rotate a view about the vertical axis.

.Rotate a view about the vertical axis.
[example]
====

mged> **knob y 1**::
Starts the view rotating about the vertical axis.
