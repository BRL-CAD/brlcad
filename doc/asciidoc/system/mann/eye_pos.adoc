:doctype: manpage
:mansource: BRL-CAD
:manmanual: BRL-CAD User Commands

== NAME
eye_pos - alias for eye; get or set the camera eye point position

== SYNOPSIS

[source]
----
eye_pos
eye_pos x y z
eye_pos "x y z"
----


[[description]]
== DESCRIPTION

The *eye_pos* command is an alias for the *eye* command. Both names invoke identical behavior: getting or setting the camera eye point position in model coordinates.

With no arguments, *eye_pos* returns the current eye point as a space-separated `x y z` triplet in the current working units.

With arguments, *eye_pos* sets the eye point to the specified position. The position may be given as three separate floating-point values or as a single quoted space-separated string.

The eye point defines where the virtual camera is positioned in model space. Moving the eye point pans and zooms the view simultaneously.

See *eye*(nged) for full documentation.

[[examples]]
== EXAMPLES

.Get the current eye point
[example]
====
mged> **eye_pos**

Returns the current camera eye position in model coordinates, for example: `0 0 1000`.
====

.Set the eye point
[example]
====
mged> **eye_pos 0 -500 500**

Positions the camera eye point at (0, -500, 500) in model coordinates.
====
