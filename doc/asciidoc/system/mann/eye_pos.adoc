== NAME
eye_pos - set eye position to given model coordinates

== SYNOPSIS

[source]
----
eye_pos mx my mz
----

[[description]]
== DESCRIPTION

The *eye_pos* command is used to set eye position to given model coordinates.

[[examples]]
== EXAMPLES

.Basic usage of the *eye_pos* command.
[example]
====
mged> **eye_pos mx my mz**

Sets the eye position to the specified model coordinates.
====
