== NAME
brep selection - Operate on components of a specified BRep solid.

== SYNOPSIS

[source]
----
brep {obj}selection [append|translate] [arguments]
----


[[description]]
== DESCRIPTION

If _obj_ is not a brep object, then *brep* will quit and report that fact. Otherwise, attempt to operate on the selected components. This command should be viewed as dangerous and experimental - it can change BRep data, but it does not guarantee preservation of solidity while doing so - if edge curve points are moved, for example, an otherwise solid BRep object will no longer be solid.


Translate a surface control vertex
