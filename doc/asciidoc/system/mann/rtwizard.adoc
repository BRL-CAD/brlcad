== NAME
rtwizard - run the rtwizard graphical interface

== SYNOPSIS

[source]
----
rtwizard [options]
----

[[description]]
== DESCRIPTION

The *rtwizard* command is used to run the rtwizard graphical interface.

[[examples]]
== EXAMPLES

.Basic usage of the *rtwizard* command.
[example]
====
mged> **rtwizard**

Runs the rtwizard graphical interface.
====
