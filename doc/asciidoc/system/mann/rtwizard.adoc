== NAME
rtwizard - launch the RtWizard GUI or render using the rtwizard subprocess

== SYNOPSIS

[source]
----
rtwizard [options]
----


[[description]]
== DESCRIPTION

The *rtwizard* command launches the `rtwizard` rendering application as a subprocess. It automatically passes the current view parameters — including view size, orientation, and eye point — to the subprocess.

When called from MGED, *rtwizard* runs in `--no_gui` (non-interactive) mode, using the current session's view state and the open database. All options are passed through to the `rtwizard` subprocess unchanged.

For interactive GUI use of RtWizard, launch the `rtwizard` application directly from the command line rather than through MGED.

NOTE: *rtwizard* runs asynchronously. Use *rtabort* to terminate an active rtwizard subprocess.

[[examples]]
== EXAMPLES

.Launch rtwizard with current view
[example]
====
mged> **rtwizard -o output.png**

Renders the current view using the `rtwizard` subprocess and writes the result to `output.png`.
====
