== NAME
label - label objects

== SYNOPSIS

[source]
----
label object(s)
----

[[description]]
== DESCRIPTION

The *label* command is used to label objects.

[[examples]]
== EXAMPLES

.Basic usage of the *label* command.
[example]
====
mged> **label model.s**

Labels model.s in the view.
====
