== NAME
opendb - Open a BRL-CAD database and make it current.

== SYNOPSIS

[source]
----
opendb
opendb [-c] [-f] [-h] [-o] filename
opendb [--create] [--flip-endian] [--help] [--open] filename
----


[[description]]
== DESCRIPTION

With no filename argument, *opendb* reports the currently open database path.
If no database is open, it reports that state instead.

When a _filename_ is supplied, *opendb* closes any currently open database and
opens the requested database file.

By default, *opendb* is willing to create a new database if the requested file
does not already exist. Use *-o* or *--open* to require that the file already
exist.

If *-f* or *--flip-endian* is supplied and the target file is a v4 database,
the file is treated as a binary-incompatible database and opened with flipped
endianness. This is useful for legacy v4 databases created on incompatible
platforms. When such flipping is applied, the database is made read-only.


[[options]]
== OPTIONS

*-c*, *--create*::
Allow creation of a new database if _filename_ does not already exist. This is
the default behavior and overrides *-o*.

*-f*, *--flip-endian*::
Treat a v4 database as a binary-incompatible file and flip its endianness on
open. Ignored for non-v4 databases.

*-h*, *--help*::
Print command help.

*-o*, *--open*::
Require that _filename_ already exist and open only that file.


[[examples]]
== EXAMPLES

.Open an existing database.
[example]
====
mged> **opendb -o model.g**

Closes the current database, if any, and opens the existing _model.g_ file.

====

.Report the currently open database path.
[example]
====
mged> **opendb**

Returns the name of the current database file.

====

.Open a legacy v4 database with forced endian flipping.
[example]
====
mged> **opendb -f legacy_v4.g**

Opens _legacy_v4.g_ as a binary-incompatible v4 database. If flipping is
applied, the database is opened read-only.

====
