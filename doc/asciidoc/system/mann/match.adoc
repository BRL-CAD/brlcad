== NAME
match - returns all database objects matching the given expression

== SYNOPSIS

[source]
----
match exp
----

[[description]]
== DESCRIPTION

The *match* command is used to return all database objects matching the given expression.

[[examples]]
== EXAMPLES

.Basic usage of the *match* command.
[example]
====
mged> **match *.s**

Returns all database objects matching the *.s expression.
====
