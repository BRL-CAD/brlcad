== NAME
Z - Clears the _mged_ display.

== SYNOPSIS

[source]
----
Z
----


[[description]]
== DESCRIPTION

Zap (i.e., clear) the _mged_ display.


[[examples]]
== EXAMPLES

Clear the _mged_ display.

.Enter *Z* to clear the _mged_ display.
[example]
====

mged> **Z**::
The _mged_ display is clear.



====
