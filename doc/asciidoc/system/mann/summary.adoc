== NAME
summary - The *summary* command with no arguments lists the number of primitive shapes, regions, and nonregion combinations in the current database.

== SYNOPSIS

[source]
----
summary
summary [prg]*
summary [-o | --obj] {object}
----


[[description]]
== DESCRIPTION

With no arguments, *summary* lists the number of primitive shapes,
regions, and non-region combinations in the current database.

The historical single-token letters `p`, `r`, and `g` remain supported
for reporting primitives, regions, and/or groups. They may be
concatenated to request combined listing(s) of primitive names,
region names, and combination names respectively.
The selector is mutually exclusive with an object name or *--obj*.

Modern usage can also summarize one specific database object either by
naming it directly as the lone argument or by supplying *-o* or
*--obj*.


[[options]]
== OPTIONS

*-h*, *-?*, *--help*::
Print help and exit.

*-o*, *--obj* _name_::
Summarize the named database object.


[[examples]]
== EXAMPLES

The example shows the use of the *summary* command with an argument to list a summary of primitive shapes and regions for the current database.

.List a summary of primitive shapes and regions for the current database.
[example]
====
mged>**summary pr**

Lists a summary of primitive shapes and regions for the current database.

====

.Summarize one object explicitly.
[example]
====
mged> **summary --obj turret.r**

Reports the summary information for _turret.r_.

====
