== NAME
fbclear - clear the active framebuffer

== SYNOPSIS

[source]
----
fbclear [rgb]
----

[[description]]
== DESCRIPTION

The *fbclear* command is used to clear the active framebuffer.

[[examples]]
== EXAMPLES

.Basic usage of the *fbclear* command.
[example]
====
mged> **fbclear 255/0/0**

Clears the framebuffer to red (255/0/0).
====
