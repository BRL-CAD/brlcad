== NAME
rpp-arch - Used to round a specified face of an RPP by creating an RCC based on the parameters of the RPP.

== SYNOPSIS

[source]
----
rpp-arch {rppname newname face}
----


[[description]]
== DESCRIPTION

Used to round a specified face of an RPP by creating an RCC based on the parameters of the RPP.


[[examples]]
== EXAMPLES

The example shows the use of the *rpp-arch* command to create an RCC shape at a specified face of the RPP.

.Create an RCC shape at a specified face of an RPP.
[example]
====
mged>**rpp-arch rpp.s arch.s 1234**

Creates an RCC shape named _arch.s_ at the 1234 face of the RPP.

====
