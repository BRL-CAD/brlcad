== NAME
reair - Set the 'aircode' attribute of all regions within a group.

== SYNOPSIS

[source]
----
reair {comb} {aircode}
----


[[description]]
== DESCRIPTION

The 'reair' command assigns the 'aircode' attribute value for all regions within a specified combination.


[[examples]]
== EXAMPLES

Set the aircode

.Set the aircode to 2.
[example]
====
mged>**reair example.c 2**

All regions under example.c will have their aircode reset to 2.

====
