== NAME
dplot - draws plotfiles from the given logfile

== SYNOPSIS

[source]
----
dplot logfile cmd [args]
----

[[description]]
== DESCRIPTION

The *dplot* command parses a log file generated by the B-Rep surface-surface intersection (SSI) routines and visually renders the geometry intersections using MGED's overlay system. This is primarily a debugging tool for analyzing NURBS intersection events.

It requires a `logfile` (usually a `.plot3` file generated by the evaluator) and a `cmd` to specify what geometry to plot. The command is stateful and will prompt the user to "Press [Enter]" to advance through sequential events.

The available `cmd` subcommands are:

* *ssx* : Show all intersecting surface pairs.
* *ssx N* : Show intersections specifically for surface-surface intersection (SSX) pair `N`.
* *isocsx N* : Show intersecting isocurve-surface pairs for SSX pair `N`.
* *isocsx N M* : Show intersections of SSX pair `N`, specifically for isocurve-surface pair `M`.
* *fcurves N* : Show clipped face curves for SSX pair `N`.
* *lcurves* : Show linked SSX curves used to split faces.
* *faces* : Show split faces used to construct the final boolean result.

[[examples]]
== EXAMPLES

.Iterate through intersecting surface pairs
[example]
====
mged> **dplot intersect.log ssx**

Parses `intersect.log` and visually overlays the first surface-surface intersection pair, prompting the user to press Enter to advance to the next pair.
====

.View clipped face curves for a specific intersection
[example]
====
mged> **dplot intersect.log fcurves 2**

Displays the clipped face curves associated with the third (0-indexed `2`) surface-surface intersection pair recorded in `intersect.log`.
====
