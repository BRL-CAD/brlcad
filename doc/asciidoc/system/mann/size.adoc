== NAME
size - Sets the size of the current viewing cube to the specified __view_size__ (in local units). Without arguments, returns the current view size.

== SYNOPSIS

[source]
----
size [view_size]
----


[[description]]
== DESCRIPTION

Sets the size of the current viewing cube to the specified view_size (in local units). This size is the length of any side of the square MGED display. If no arguments are supplied, returns the current view size. See also the *view* command.


[[examples]]
== EXAMPLES

The example shows the use of the *size* command to set the MGED display to a specified number of units across.

.Set the size of the MGED display.
[example]
====
mged>**size 250**

Sets the MGED display to be 250 units across.

====
