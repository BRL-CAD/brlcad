== NAME
annotate - attach annotation objects to geometry

== SYNOPSIS

[source]
----
annotate [object(s)] [-n name] [-p x y z]
----


[[description]]
== DESCRIPTION

The *annotate* command creates or manages annotation objects associated with the specified geometry `objects`. Annotations are non-geometry database objects that attach labels, notes, leaders, or dimension indicators to model elements.

If no objects are given, the command displays usage information.

Annotation types supported include:

* *label* / *leader* — text label with optional leader line
* *angular* / *radial* / *dimension* — dimensional annotations
* *note* — free-form note
* *box* — boxed text region
* *axes* / *plane* — coordinate system or plane indicators
* *table* — tabular annotation data

Annotations can be aligned to the view (always facing the screen) or to the model (fixed in 3D space).

=== Options

*-n name*::
Assign the specified `name` to the new annotation object instead of using an automatically generated name.

*-p x y z*::
Place the annotation at model coordinates `(x, y, z)`.

[[examples]]
== EXAMPLES

.Annotate an object with a label
[example]
====
mged> **annotate hull.r -n hull_note -p 0 0 500**

Creates a label annotation named `hull_note` associated with `hull.r`, positioned at model coordinate (0, 0, 500).
====
