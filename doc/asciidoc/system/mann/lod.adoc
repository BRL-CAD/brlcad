== NAME
lod - configure Level of Detail (LOD) drawing

== SYNOPSIS

[source]
----
lod
lod [--help]
lod [0|1]
lod csg|mesh [0|1]
lod cache [clear [all_files]|exists]
lod scale|point_scale|curve_scale [factor]
lod bot_threshold [face_count]
view lod [same arguments]
----

[[description]]
== DESCRIPTION

The *lod* command configures the Level of Detail (LoD) drawing behavior for the current view. *view lod* is an equivalent spelling. When LoD drawing is enabled, BRL-CAD adapts the rendering complexity of wireframe geometry based on view conditions, improving performance for complex models.

When called with no arguments, *lod* prints the current LoD configuration, including:

* Whether mesh and CSG adaptive LoD drawing are enabled.
* The current overall, point, and curve scale factors.
* The current BoT face threshold (the number of faces at which a BoT switches to simplified display).

=== Subcommands

* *1* : Enable both mesh and CSG adaptive LoD drawing.
* *0* : Disable both mesh and CSG adaptive LoD drawing.
* *csg [0|1]* : Query or set CSG adaptive LoD drawing.
* *mesh [0|1]* : Query or set mesh adaptive LoD drawing.
* *scale [factor]* : Query or set the overall LoD scale.
* *point_scale [factor]* : Query or set the point-density scale.
* *curve_scale [factor]* : Query or set the curve-complexity scale.
* *bot_threshold [face_count]* : Query or set the nonnegative BoT face threshold.
* *cache* : Build the current database's mesh LoD cache.
* *cache clear* : Clear the in-memory cache.
* *cache clear all_files* : Clear cache data for all files.
* *cache exists* : Test whether all BoT and BRep entries have cached LoD data.

All scale values must be finite numbers. The command rejects an unknown action, an invalid binary setting, a negative threshold, or extra action arguments before changing view state.

[[examples]]
== EXAMPLES

.Query current LoD state
[example]
====
mged> **lod**

----
enabled(mesh/csg): 1/1
scale: 1
point_scale: 1
curve_scale: 1
bot_threshold: 32768
----

Prints the current LoD configuration.
====

.Enable LoD and reduce curve complexity
[example]
====
mged> **view lod 1**
mged> **view lod curve_scale 0.5**

Enables LoD drawing and halves the curve complexity scale, reducing the number of line segments used to approximate curves.
====

.Query a single LoD setting
[example]
====
mged> **lod mesh**

Returns `1` if mesh LoD drawing is enabled and `0` otherwise. Useful in scripts or Tcl procedures.
====
