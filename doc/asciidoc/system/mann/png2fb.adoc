== NAME
png2fb - send PNG (Portable Network Graphics) files to a framebuffer

== SYNOPSIS

[source]
----
png2fb [options]
----

[[description]]
== DESCRIPTION

The *png2fb* command is used to send PNG (Portable Network Graphics) files to a framebuffer.

[[examples]]
== EXAMPLES

.Basic usage of the *png2fb* command.
[example]
====
mged> **png2fb**

Sends a PNG file to the framebuffer.
====
