== NAME
list - list objects and elements in the database

== SYNOPSIS

[source]
----
list [-r] [-t] [-v] <objects>
----

[[description]]
== DESCRIPTION

The *list* command is used to list objects and elements in the database.

[[examples]]
== EXAMPLES

.Basic usage of the *list* command.
[example]
====
mged> **list -r -t -v model.s**

Lists objects and elements for model.s.
====
