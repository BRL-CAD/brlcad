== NAME
dbfind - Displays all combinations that have any of the _objects_ specified as a member.

== SYNOPSIS

[source]
----
dbfind {objects}
----


[[description]]
== DESCRIPTION

Displays all combinations that have any of the _objects_ specified as a member.


[[examples]]
== EXAMPLES

The example shows the use of the *dbfind* command to display all combinations that refer to the specified object.

.Display all combinations that refer to a specified object.
[example]
====

mged> **dbfind shapea**::
Lists all combinations that refer to _shapea_.
