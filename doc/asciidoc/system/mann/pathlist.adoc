== NAME
pathlist - Lists all existing paths that start from the specified combinations and end at a primitive shape.

== SYNOPSIS

[source]
----
pathlist [-n|--noleaf] combination
----


[[description]]
== DESCRIPTION

Lists all existing paths that start from the specified combination and end at a
primitive shape.  The *-n* or *--noleaf* option omits the leaf object from
each reported path.  The historical *-noleaf* spelling remains accepted for
compatibility.


[[examples]]
== EXAMPLES

The example shows the use of the *pathlist* command to list all paths that start from the given combinations and end at a primitive shape.

.List all paths that start from given combinations and end at a primitive shape.
[example]
====
mged>**pathlist group1**

Lists all existing paths that start from _group1_ and end at primitive shapes.

====
