== NAME
clear - Clears the _mged_ command window.

== SYNOPSIS

[source]
----
clear
----


[[description]]
== DESCRIPTION

Clear the _mged_ command window.


[[examples]]
== EXAMPLES

Clear the _mged_ command window.

.Enter *clear* to clear the _mged_ command window.
[example]
====

mged> **clear**::
The _mged_ command window is clear.
