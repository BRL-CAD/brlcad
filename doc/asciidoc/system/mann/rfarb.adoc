== NAME
rfarb - Creates a new ARB8 shape based on rotation and fallback angles.

== SYNOPSIS

[source]
----
rfarb name px py pz rotation_angle fallback_angle coord1 u1 v1 coord2 u2 v2 coord3 u3 v3 thickness
----


[[description]]
== DESCRIPTION

Creates a new ARB8 shape based on rotation and fallback angles.  _px_, _py_,
and _pz_ specify one corner of the first face.  For each of the other three
face vertices, _coord1_, _coord2_, and _coord3_ select the coordinate to solve
(_x_, _y_, or _z_), while the following _u_ and _v_ values provide the other
two coordinates.  The missing coordinate is calculated so that the face is
planar.  _thickness_ must be nonzero; the first face is extruded normally by
that amount to complete the ARB8.


[[examples]]
== EXAMPLES

The example shows the use of the *rfarb* command to create a new ARB8 shape.

.Create a new ARB8 shape.
[example]
====
mged>**rfarb wedge.s 0 0 0 45 30 x 1 1 y 1 1 z 1 1 1**

Creates the ARB8 _wedge.s_ using a point at the origin, the specified plane
angles, three planar face vertices, and a thickness of 1.
