== NAME
arb - Performs operations on ARB shapes, such as creation and repair.

== SYNOPSIS

[source]
----
arb create {arb_name} {rotation} {fallback}

arb repair [-h] [-t tol] [-o output_name] {arb_name} [arb_name...]
----


[[description]]
== DESCRIPTION

The *arb* command performs operations on ARB shapes depending on the subcommand specified.

*create*::
Creates a new ARB shape with the specified __arb_name__. The new ARB will be 20 inches by 20 inches and 2 inches thick. The square faces will be perpendicular to the direction defined by the rotation and fallback angles. This direction can be determined by interpreting the rotation angle as an azimuth and the fallback angle as an elevation as in the *ae* command. (Note: the legacy syntax `arb arb_name rotation fallback` is deprecated and will be removed in a future release).

*repair*::
Attempts to repair and canonicalize an existing ARB primitive. If `-o output_name` is provided, the repaired ARB is saved as a new object; otherwise, the repair happens in-place. A distance tolerance for snapping vertices can be supplied via the `-t` option. If omitted, the default BRL-CAD editing tolerance is used.


[[examples]]
== EXAMPLES

The first example shows the use of the *arb* command to create a new ARB shape with a specified name. The second example shows the use of the *ae* command to determine the view orientation of the arb. The third example demonstrates repairing a degenerate ARB.

.Create __new_arb__ with a specific rotation angle and fallback angle.
[example]
====
mged>**arb create new_arb 35 25**

Create new_arb with a rotation angle of 35° and a fallback angle of 25°.

====

.Using the *ae* command to determine the view orientation of the arb
[example]
====
mged>**ae 35 25**

Rotates view to look straight on at square face of __new_arb__.

====

.Repair an existing ARB and save it to a new object
[example]
====
mged>**arb repair -o fixed_arb broken_arb**

Attempts to canonicalize and repair `broken_arb`, and saves the result to `fixed_arb`.

====

.Repair an ARB using a tight tolerance
[example]
====
mged>**arb repair -t 1e-7 broken_arb**

Attempts to canonicalize and repair `broken_arb` in-place, using a tight `1e-7` tolerance to strictly enforce planar faces (matching the default tolerance of the `lint` command).

====
