== NAME
group - group the specified objects into a combination

== SYNOPSIS

[source]
----
group gname object(s)
----

[[description]]
== DESCRIPTION

The *group* command is used to group the specified objects into a combination.

[[examples]]
== EXAMPLES

.Basic usage of the *group* command.
[example]
====
mged> **group new_group.c model.s**

Groups model.s into the combination new_group.c.
====
