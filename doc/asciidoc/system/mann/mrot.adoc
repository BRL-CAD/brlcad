== NAME
mrot - Rotates the view using model _x y z_.

== SYNOPSIS

[source]
----
mrot {x y z}
----


[[description]]
== DESCRIPTION

Rotates the view using model _x y z_.


[[examples]]
== EXAMPLES

The example shows the use of the *mrot* command to rotate the view using model _x y z_.

.Rotate the view using model _x y z_.
[example]
====
mged>**mrot 0 0 10**

Rotates the view about the model _z_ axis by 10°.

====
