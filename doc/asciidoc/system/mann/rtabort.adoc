== NAME
rtabort - abort the associated raytraces

== SYNOPSIS

[source]
----
rtabort [options]
----

[[description]]
== DESCRIPTION

The *rtabort* command is used to abort the associated raytraces.

[[examples]]
== EXAMPLES

.Basic usage of the *rtabort* command.
[example]
====
mged> **rtabort**

Aborts the associated raytraces.
====
