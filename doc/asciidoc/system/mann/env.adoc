== NAME
env - Print the current environmental variables.

== SYNOPSIS

[source]
----
env [options] [pattern ...]
env get variable
env set variable value
----


[[description]]
== DESCRIPTION

Without a subcommand, *env* lists the BRL-CAD environment variables that are
currently set.  Optional shell-style patterns limit the listed variable names.

*env get* reports one known variable's current value.  *env set* changes one
known variable for the current process and reports the new value.

== OPTIONS

*-h, --help*::
Print command usage.

*-A, --all*::
List known variables even when they are not set.

*-S, --system*::
Include known variables used by external system or third-party software.

*-C, --context*::
Show the BRL-CAD library or executable associated with each listed variable.

*-M, --memory*::
Report the current memory totals and page size instead of listing variables.


[[examples]]
== EXAMPLES

List variables associated with BRL-CAD:

.Enter *env* to display the current environmental variables.
[example]
====

mged> **env**::
The current environmental variables and their values are displayed.

mged> **env --all BRLCAD_***::
All known BRL-CAD-prefixed variables are listed, including unset ones.
