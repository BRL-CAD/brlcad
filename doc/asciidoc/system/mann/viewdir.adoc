== NAME
viewdir - Returns the direction vector using the current azimuth and elevation (AE).

== SYNOPSIS

[source]
----
viewdir [-i]
----


[[description]]
== DESCRIPTION

Returns the direction vector using the current azimuth and elevation as input. If the _-i_ option is used the commmand will return the inverse directional vector. See also the *ae2dir*, *dir2ae* and *qvrot* commands.


[[examples]]
== EXAMPLES

The example shows the use of the *viewdir* command to get the direction vector.

.Find the current direction vector.
[example]
====
mged>**viewdir**

Returns the direction vector using the current azimuth and elevation.

====
