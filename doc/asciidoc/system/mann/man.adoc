== NAME
man - Opens the graphical man page viewer.

== SYNOPSIS

[source]
----
man [cmdname]
----


[[description]]
== DESCRIPTION

Launches the *brlman* graphical man page viewer with the man page documenting the specified MGED command name. If no command name is provided, The graphical man page viewer will open the entry for the first command (or the last command looked at if the manual was previously opened in the current MGED session).
