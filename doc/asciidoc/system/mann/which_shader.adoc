== NAME
which_shader - Lists all the regions that use one of the _shaders_ specified.

== SYNOPSIS

[source]
----
which_shader [-s] {shaders}
----


[[description]]
== DESCRIPTION

Lists all the regions that use one of the _shaders_ specified.

If *-s* is supplied, the output is formatted more compactly for scripting.


[[options]]
== OPTIONS

*-s*::
Use compact script-oriented output formatting.


[[examples]]
== EXAMPLES

The example shows the use of the *which_shader* command to list all the regions that use the specified shaders.

.List the regions that use the specified shaders.
[example]
====
mged>**which_shader plastic light**

Lists all regions in the current database that use the _plastic_ or _light_ shaders.
