== NAME
relos - Sets the LOS percentage for all regions found in the given group.

== SYNOPSIS

[source]
----
relos {assembly} {value}
----


[[description]]
== DESCRIPTION

The *relos* sets the LOS value for all regions found under the given assembly to the inputted value. If a region is given, it will set the LOS for just that region.


[[examples]]
== EXAMPLES

Set the LOS

.Set the LOS to 50.
[example]
====
mged>**relos example.c 50**

All regions under example.c will have their LOS value reset to 50.

====
