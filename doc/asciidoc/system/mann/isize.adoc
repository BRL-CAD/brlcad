== NAME
isize - set or get the inverse of the view size

== SYNOPSIS

[source]
----
isize [options]
----

[[description]]
== DESCRIPTION

The *isize* command is used to set or get the inverse of the view size.

[[examples]]
== EXAMPLES

.Basic usage of the *isize* command.
[example]
====
mged> **isize**

Sets or gets the inverse of the view size.
====
