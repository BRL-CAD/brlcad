== NAME
isize - return the inverse of the current view size

== SYNOPSIS

[source]
----
isize
----


[[description]]
== DESCRIPTION

The *isize* command returns the reciprocal of the current view size in the current working units. This is a read-only getter — no arguments are accepted.

The inverse view size is related to the current zoom level. It equals `1 / view_size`, where `view_size` is the value returned by the *size* command. A larger `isize` value corresponds to a more zoomed-in view (smaller view size); a smaller `isize` value corresponds to a more zoomed-out view.

This value is primarily useful in scripts that compute view-space to model-space scaling factors.

[[examples]]
== EXAMPLES

.Get the inverse view size
[example]
====
mged> **isize**

Returns the reciprocal of the current view size in working units, for example: `0.001` when the view size is 1000 mm.
====

.Use isize in a calculation
[example]
====
mged> **set inv [isize]**
mged> **set sz [size]**
mged> **expr {$inv * $sz}**

Retrieves both `isize` and `size` and verifies their product equals 1.0.
====
