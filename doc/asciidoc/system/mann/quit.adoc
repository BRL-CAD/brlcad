== NAME
quit - Ends the MGED process.

== SYNOPSIS

[source]
----
quit
----


[[description]]
== DESCRIPTION

Ends the MGED process.

MGED does not queue edits for a later save step. Changes are written to the
database as they are made, so *quit* does not restore the database to an
earlier pre-edited state.

MGED also accepts *q* and *exit* for the same operation.


[[examples]]
== EXAMPLES

The example shows the use of the *quit* command to end the current MGED session.

.End the current MGED session.
[example]
====
mged> **quit**

Quits the current MGED session.

====
