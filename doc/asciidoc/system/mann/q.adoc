== NAME
q - Short command name for *quit*.

== SYNOPSIS

[source]
----
q
----


[[description]]
== DESCRIPTION

The *q* command shares its core implementation with *quit*.
It ends the current MGED session.

See *quit* for the detailed behavior.


[[examples]]
== EXAMPLES

The example shows the use of the *q* command to quit a current MGED session.

.Quit MGED using the short command.
[example]
====
mged> **q**

Performs the same operation as *quit*.

====
