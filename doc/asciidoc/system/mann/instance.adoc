== NAME
instance - instantiate an object

== SYNOPSIS

[source]
----
instance obj comb [op]
----

[[description]]
== DESCRIPTION

The *instance* command is used to instantiate an object.

[[examples]]
== EXAMPLES

.Basic usage of the *instance* command.
[example]
====
mged> **instance model.s group.c +**

Instantiates model.s into group.c with the + operator.
====
