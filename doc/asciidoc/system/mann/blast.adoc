:doctype: manpage

== NAME
blast, B - clear the display and draw the specified objects

== SYNOPSIS

[source]
----
blast [-C r/g/b] object(s)
B [-C r/g/b] object(s)
----


[[description]]
== DESCRIPTION

The *blast* command (also available as *B*) erases all currently displayed objects and then draws the specified `objects`. It is equivalent to running *zap* followed by *draw* with the same arguments.

All options accepted by *draw* are also accepted by *blast*, since it passes its arguments directly to the draw command after clearing the display. This includes attribute-based selection, color overrides, and BOT display modes.

At least one object name must be provided.

=== Options

*-C r/g/b*::
Override the display color for all drawn objects using the specified RGB triple (each 0–255), separated by `/`. This overrides any color assigned to the objects in the database.

[[examples]]
== EXAMPLES

.Clear display and draw a single object
[example]
====
mged> **blast hull.c**

Erases everything currently shown and draws `hull.c`.
====

.Clear and draw multiple objects
[example]
====
mged> **B engine.c wheels.c**

Erases the current display and draws both `engine.c` and `wheels.c`. Uses the short `B` alias.
====

.Clear and draw with color override
[example]
====
mged> **blast -C 255/0/0 target.c**

Erases the display and draws `target.c` with an RGB override color of red (255,0,0), overriding all other color specifications.
====
