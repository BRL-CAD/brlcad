== NAME
blast - clear screen, draw objects

== SYNOPSIS

[source]
----
blast [-C#/#/#] object(s)
----

[[description]]
== DESCRIPTION

The *blast* command is used to clear screen, draw objects.

[[options]]
== OPTIONS

*-C*::
Specify a color to draw the objects with, formatted as R/G/B (e.g. 255/0/0).

[[examples]]
== EXAMPLES

.Basic usage of the *blast* command.
[example]
====
mged> **blast -C255/0/0 group.c**

Clears the screen and draws group.c with the color 255/0/0.
====
