== NAME
coil - make a coil shape

== SYNOPSIS

[source]
----
coil [options] object
----

[[description]]
== DESCRIPTION

The *coil* command is used to make a coil shape.

[[examples]]
== EXAMPLES

.Basic usage of the *coil* command.
[example]
====
mged> **coil model.s**

Makes a coil shape from model.s.
====
