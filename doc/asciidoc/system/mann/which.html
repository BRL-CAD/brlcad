WHICH(nged) Manual Page

NAME

which - alias for whichid; list regions using specified region identifier(s)

SYNOPSIS

which [options] region_id(s)

DESCRIPTION

The which command lists all regions that use the specified region_id(s). This command is an alias for the whichid command. ID numbers can be specified individually or as an inclusive range using either start-end or start:end (for example, 100-200 or 100:200).

OPTIONS

-h, -?, --help

Print help and exit.

-s, --script

Use a space-separated output format that is easier to parse in scripts.

-V

List all active IDs in the given range, even if no associated regions are found.

-U, --unused

Report unused IDs in the specified range.

--root root_name

Search only in the tree below the specified root_name.

EXAMPLES

Example 1. Basic usage of the which command.

mged> which 1000

Lists all regions in the database that have a region ID of 1000.
