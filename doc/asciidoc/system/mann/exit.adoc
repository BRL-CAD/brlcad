== NAME
exit - Ends the MGED process.

== SYNOPSIS

[source]
----
exit
----


[[description]]
== DESCRIPTION

Ends the MGED process. It is a synonym for the *quit* command.


[[examples]]
== EXAMPLES

The example shows the use of the *exit* command to stop MGED.

.Using the *exit* command to end MGED.
[example]
====

mged> **exit**::
MGED stops.



====
