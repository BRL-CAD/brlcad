= GRID(nged)
:doctype: manpage
:mansource: BRL-CAD
:manmanual: BRL-CAD User Commands

== NAME
grid - control the display grid and snap-to-grid behavior

== SYNOPSIS

[source]
----
grid parameter [value ...]
----


[[description]]
== DESCRIPTION

The *grid* command controls the view grid: a regular overlay of horizontal and vertical lines drawn on the display to aid in alignment and measurement. It also controls snap-to-grid behavior, which constrains view center movement to the nearest grid intersection.

The `parameter` selects a grid attribute or action for the current view. If a
value is provided it sets that attribute; if omitted the current value is
returned. `help` prints the command syntax and `vars` prints every current
grid value.

=== Parameters

[cols="1,1,4"]
|===
|Parameter |Value |Description

|`color`
|`r/g/b`, `r,g,b`, `r;g;b`, or `r g b`
|Get or set the grid line color as three integers (0–255). With no value, returns current color.

|`anchor`
|`x/y/z`, `x,y,z`, `x;y;z`, or `x y z`
|Get or set the grid anchor in current working units. With no value, returns the current anchor.

|`draw`
|`0` or `1`
|Enable (`1`) or disable (`0`) grid display. With no value, returns current draw state.

|`snap`
|`0` or `1`
|Enable (`1`) or disable (`0`) snap-to-grid. With no value, returns current snap state.

|`rh`
|float
|Get or set the horizontal grid resolution (spacing) in the current working units.

|`rv`
|float
|Get or set the vertical grid resolution (spacing) in the current working units.

|`mrh`
|integer
|Get or set the major grid line interval (horizontal): a major line is drawn every N minor lines.

|`mrv`
|integer
|Get or set the major grid line interval (vertical).

|`vsnap`
|(none)
|Snap the current view center to the nearest grid intersection. Takes no value argument.

|`vars`
|(none)
|Print all current grid values.

|`help`
|(none)
|Print grid command syntax.

|===

[[examples]]
== EXAMPLES

.Enable the grid
[example]
====
mged> **grid draw 1**

Turns on grid display for the current view.
====

.Set grid spacing to 10 mm
[example]
====
mged> **grid rh 10**
mged> **grid rv 10**

Sets horizontal and vertical grid spacing to 10 units in the current working units.
====

.Enable snap-to-grid
[example]
====
mged> **grid snap 1**

Enables snap-to-grid for the current view. The view center will snap to the nearest grid point during interactive navigation.
====

.Set grid color to green
[example]
====
mged> **grid color 0;200;0**

Sets the grid line color to green (R=0, G=200, B=0).
====

.Get the current horizontal resolution
[example]
====
mged> **grid rh**

Returns the current horizontal grid resolution in working units, for example: `10`.
====

.Snap view center to nearest grid point
[example]
====
mged> **grid vsnap**

Moves the view center to the nearest grid intersection.
====
