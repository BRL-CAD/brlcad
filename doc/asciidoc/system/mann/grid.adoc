== NAME
grid - convert grid xy to view coordinates (local units)

== SYNOPSIS

[source]
----
grid x y
----

[[description]]
== DESCRIPTION

The *grid* command is used to convert grid xy to view coordinates (local units).

[[examples]]
== EXAMPLES

.Basic usage of the *grid* command.
[example]
====
mged> **grid 10 20**

Converts grid (10, 20) to view coordinates.
====
