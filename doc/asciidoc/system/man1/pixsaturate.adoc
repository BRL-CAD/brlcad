== NAME
pixsaturate - saturate a pix file

== SYNOPSIS

[source]
----
pixsaturate saturation < infile.pix > outfile.pix
----


[[description]]
== DESCRIPTION

*pixsaturate* saturates a color pix file using the *saturation* value. A value of 0 gives monochrome, 1.0 gives the original image, and values larger than 1.0 give a more saturated image.
