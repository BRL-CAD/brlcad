== NAME
dauto - compute the unbiased autocorrelation function of a stream of doubles

== SYNOPSIS

[source]
----
dauto [window_size] < input.doubles > output.doubles
----


[[description]]
== DESCRIPTION

*dauto* reads a stream of double-precision floating-point numbers from
standard input, computes an unbiased estimate of the autocorrelation
function over successive fixed-size windows, and writes the resulting
double-precision values to standard output.

Input is processed in non-overlapping records of _window_size_ samples
(default 512). For each record of _L_ samples, *dauto* estimates the
autocorrelation at lags 0 through _L_-1, writing _L_ values per record.
The estimate at lag 0 is based on all _L_ samples and the estimate at
lag _L_-1 on only one; zeros are assumed outside the record. The
estimate is unbiased by scaling each lag by a factor inversely
proportional to the number of contributing samples. A final partial
record shorter than _window_size_ is zero-padded.

Both standard input and standard output must be redirected; *dauto*
exits with a usage message if either is a terminal. See *dauto2*(1) for
a biased variant.

[[options]]
== OPTIONS

_window_size_::
  Number of samples per record, controlling the maximum lag computed.
  Must be a positive integer. Defaults to 512 if omitted.

[[examples]]
== EXAMPLES

.Autocorrelate a sampled signal
====
[source,bash]
----
dauto 1024 < signal.doubles > autocor.doubles
----
Computes the unbiased autocorrelation of a stream of double-precision
values in 1024-sample windows. The resulting stream of doubles can be
fed to *dpeak*(1) to locate dominant periodicities in the signal.
====
