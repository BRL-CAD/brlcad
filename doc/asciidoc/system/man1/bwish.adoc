== NAME

bwish - Tcl/Tk-based interactive windowing shell for BRL-CAD

== SYNOPSIS

*bwish* [_script_] [_arg_...]

== DESCRIPTION

*bwish* is a Tcl/Tk-based interactive windowing shell for BRL-CAD. It operates similarly to the standard Tk *wish*(1) command, but is pre-configured to initialize BRL-CAD's specific Tcl extensions and libraries. This makes *bwish* well-suited for developing custom graphical interfaces, interactive scripting, and driving BRL-CAD tasks that require visual windows or dialogs.

When started without arguments (or if the first argument begins with a `-`), *bwish* runs interactively. It reads commands from standard input, providing command-line editing and history, and automatically evaluates the `~/.bwishrc` (or `~/.bwishrc.tcl` on Windows) file if it exists.

If a _script_ file is provided as the first argument, *bwish* evaluates the contents of that file instead of opening an interactive prompt. Any subsequent arguments are made available to the Tcl script via the global `argv` and `argc` variables.

== OPTIONS

_script_::
A Tcl/Tk script to execute. If specified as the first argument, *bwish* evaluates the script and exits when the script terminates.

_arg_...::
Optional arguments passed to the _script_. These are accessible within Tcl via the standard global `argv` list and `argc` count.

== EXAMPLES

.Running a Tk script with BRL-CAD extensions
====
Execute a Tcl/Tk script using *bwish* and pass arguments to it:

[source,bash]
----
$ bwish draw_ui.tcl 800x600
----
====

.Interactive usage
====
Launch the interactive graphical shell, which will automatically source `~/.bwishrc` if it exists:

[source,bash]
----
$ bwish
% pack [button .b -text "Exit" -command exit]
----
====
