== NAME
ap203e2-g - convert a STEP AP203 edition 2 file to a BRL-CAD geometry database

== SYNOPSIS

[source]
----
ap203e2-g [-v] [-D] [-S _summary_file_] [-o _outfile.g_ | -O _outfile.g_] _infile.stp_
----


[[description]]
== DESCRIPTION

*ap203e2-g* reads an ISO 10303-203 edition 2 (AP203e2) STEP (Standard for the
Exchange of Product model data) file and translates its boundary representation
(B-rep) geometry and assembly structure into a BRL-CAD `.g` database.

The converter maps STEP manifold solids and bounded surfaces into BRL-CAD
Boundary Representation (*brep*) objects, and preserves the assembly/part
hierarchy using BRL-CAD combinations. It shares its implementation with
*step-g*(1) but is built against the AP203 edition 2 schema. If _infile.stp_ is
given as `-`, the STEP data is read from standard input.

Progress, load timing, and convert timing are reported on standard error.

[[options]]
== OPTIONS

*-o* _outfile.g_::
  Name the BRL-CAD geometry database to create. If the file already exists,
  *ap203e2-g* refuses to overwrite it and exits with an error.

*-O* _outfile.g_::
  Name the BRL-CAD geometry database to create, forcibly overwriting it if it
  already exists.

*-S* _summary_file_::
  Write a summary log of the STEP translation to the named file.

*-v*::
  Enable verbose diagnostic output during parsing and conversion.

*-D*::
  Perform a dry run: load and parse the STEP file and simulate conversion, but
  do not write the output `.g` file. Useful for validating a STEP file or
  timing the parsing phase.

[[examples]]
== EXAMPLES

.Convert and inspect an AP203e2 part
====
[source,bash]
----
ap203e2-g -o widget.g widget.stp
mged widget.g 'tops'
----
Translates the AP203 edition 2 file _widget.stp_ into _widget.g_, then lists the
top-level objects of the resulting database with *mged*(1) so the imported
geometry can be examined or rendered.
====

.Validate a STEP file without writing output
====
[source,bash]
----
ap203e2-g -v -D -O /dev/null model.stp
----
Loads and parses _model.stp_ with verbose diagnostics enabled to confirm the
file converts cleanly, without producing a geometry database.
====
