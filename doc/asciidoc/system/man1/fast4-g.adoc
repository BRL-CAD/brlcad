== NAME
fast4-g - FASTGEN4 Translator (FASTGEN4 to BRL-CAD)

== SYNOPSIS

[source]
----
fast4-g [-d] [-w] [-q] [-o plot_file] [-m muves_warnings] [-c id_list] [-C color_file] [-x RT_DEBUG_FLAG] [-b BU_DEBUG_FLAG] [-X NMG_DEBUG_FLAG] [-h] [-?] file.fastgen4 file.g
----

[[description]]
== DESCRIPTION

*fast4-g* converts a FASTGEN 4.1 format file (_file.fastgen4_) to
a BRL-CAD geometry database (_file.g_).

The supported FASTGEN element types and their BRL-CAD equivalents
are: CCONE1/CCONE2/CCONE3 to TRC, CHEX2 to ARB8, CLINE to cline,
CQUAD/CTRI/CHEX1 to BOT faces, and CSPHERE to SPH. Region ident
numbers are computed as `group * 1000 + component`, and object
names derive from $NAME elements using an algorithm that ensures
uniqueness.

For FASTGEN2 or FASTGEN3 files, convert them using *rpatch*(1) and
then *patch-g*(1).

[[options]]
== OPTIONS

*-d*::
Print debugging information.

*-w*::
Print warnings when an object is assigned a default name.

*-q*::
Quiet mode; suppress all output except errors.

*-o* _plot_file_::
Write all CTRI and CQUAD elements to _plot_file_ in libplot3
format. The file can be viewed in MGED using the _overlay_ command.

*-m* _muves_warnings_::
Write CHGCOMP and CBACKING element information to
_muves_warnings_.

*-c* _id_list_::
Convert only the specified components. _id_list_ is a
comma-separated list of ident numbers and ranges (e.g.,
`3002,5678,1002-1100`).

*-C* _color_file_::
Read region colors from _color_file_. Each line contains five
integers: low high r g b.

*-x* _RT_DEBUG_FLAG_::
Set the librt debug flag (hex value).

*-b* _BU_DEBUG_FLAG_::
Set the libbu debug flag (hex value).

*-X* _NMG_DEBUG_FLAG_::
Set the NMG debug flag (hex value).

*-h, -?*::
Print a usage message and exit.

[[examples]]
== EXAMPLES

.Basic conversion
====
[source,bash]
----
$ fast4-g sample.fastgen4 sample.g
----
====

.Convert specific components with verbose output
====
[source,bash]
----
$ fast4-g -d -w -c 3002,5678,1002-1100 vehicle.fg4 vehicle.g
----
Converts only the specified component ident ranges, with debugging
and default-name warnings enabled.
====
