== NAME
bwfilter - apply a 3x3 kernel filter to a black and white bw image

== SYNOPSIS

[source]
----
bwfilter [-v] [-f _type_] [-d _div_] [-O _offset_] [-s _squaresize_] [-w _width_] [-n _height_] [-o _out_file.bw_] [_file.bw_]
----

[[description]]
== DESCRIPTION

*bwfilter* convolves a 3x3 kernel filter over a black and white *bw*(5)
format image.  It reads from _file.bw_ or from standard input when no
file is given, and writes the filtered result to standard output (or to
the file named with *-o*).  The outer rows and columns of the image are
left untouched.

By default the input is assumed to be 512x512 pixels and a low pass
filter is applied.  Each built-in filter has an associated divisor that
scales the convolved result back into the 0-255 range and, for some
filters, an offset that recenters the result; both may be overridden.

[[options]]
== OPTIONS

*-f* _type_::
  Select the filter to apply (default *lo*).  Recognized types are:
+
--
*lo*;; low pass
*hi*;; high pass
*vg*;; vertical gradient
*hg*;; horizontal gradient
*la*;; laplacian
*b*;; boxcar average (low pass)
--

*-d* _div_::
  Override the kernel divisor.  Most filters divide the result of the
  kernel multiplication by a fixed value to scale it into a 0-255 range
  (for example, the boxcar filter divides by nine).  A divisor of zero
  is rejected.

*-O* _offset_::
  Override the kernel offset.  Some filters add an offset to their
  results; the laplacian, for instance, is centered about zero and adds
  128 to yield a difference from neutral grey.

*-s* _squaresize_::
  Set both the width and height of the input image to _squaresize_.

*-w* _width_::
  Set the width of the input image to _width_ (default 512).

*-n* _height_::
  Set the height of the input image to _height_ (default 512).

*-o* _out_file.bw_::
  Write the filtered output to _out_file.bw_.  If standard output is
  also non-interactive (redirected or piped), the same output is
  written to both destinations.

*-v*::
  Verbose: print the kernel values and the maximum and minimum output
  values to standard error.

[[examples]]
== EXAMPLES

.Sharpen an image with a high pass filter
====
[source,bash]
----
bwfilter -f hi -s 1024 image.bw > sharp.bw
----
Applies a high pass filter to the 1024x1024 image _image.bw_ and
redirects the result to _sharp.bw_, accentuating edges and fine detail.
====

.Detect edges and view the result
====
[source,bash]
----
bwfilter -f la moon.bw | bw-pix | pix-fb
----
Runs a laplacian edge detector over _moon.bw_, converts the resulting
black and white image to *pix*(5) with *bw-pix*(1), and displays it on a
framebuffer with *pix-fb*(1) without writing any intermediate file.
====
