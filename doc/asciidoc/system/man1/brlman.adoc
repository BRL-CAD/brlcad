== NAME
brlman - manual page viewer for BRL-CAD

== SYNOPSIS

[source]
----
brlman [--gui|--no-gui] [-L language] [-S section] [-k words | -K words] [manual_page [section]]
brlman [-h|--help]
----


[[description]]
== DESCRIPTION

*brlman* is a manual page viewer for BRL-CAD programs, libraries, file
formats, and *mged*(1)/GED commands. On Unix-like systems it wraps the system
*man*(1) command, supplying the paths needed to locate BRL-CAD's installed man
pages. On systems lacking a *man* command, or when the GUI is explicitly
requested, *brlman* launches a graphical viewer that displays HTML versions of
the same documentation. The terminal and HTML pages are generated from the
same source and contain identical information.

The page to display is given by name. An optional one-character _section_ may
be supplied either as a positional argument or via *-S*. If no section is
given, *brlman* searches sections 1, 3, 5, and n in that order and shows the
first match. With no arguments at all, the graphical viewer (where available)
opens to a browsable index.

The *-k* and *-K* options search the installed BRL-CAD manual corpus instead
of opening a page. Positional words after either option extend its query.


[[options]]
== OPTIONS

*-h*, *--help*::
  Print a usage summary and exit.

*-g*, *--gui*::
  Force the graphical man page viewer, even where the terminal *man* command
  is available.

*--no-gui*::
  Disable the graphical viewer. If the terminal *man* command is unavailable,
  *brlman* will fail. Primarily useful in scripting.

*-L* _lg_, *--language* _lg_::
  Lower-case ISO 639-1 two-letter language code (e.g. *en*) used when
  selecting man pages. If not given, *brlman* uses the first two characters of
  the *LANG* environment variable, falling back to *en*.

*-S* _#_, *--section* _#_::
  Section to search. Useful when a program and a GED command share a name; the
  default page is the program (section 1), so *-S n* selects the GED command
  page instead. Allowed sections are 1 (programs), 3 (libraries), 5
  (conventions/formats), and n (GED commands).

*-k* _words_, *--keyword* _words_::
  Search manual-page names, synopses, and brief descriptions.

*-K* _words_, *--full-text* _words_::
  Search full manual-page text. This is mutually exclusive with *-k*.

[[examples]]
== EXAMPLES

.Viewing a program manual page
====
[source,bash]
----
brlman mged
----
Opens the manual page for the *mged*(1) command, defaulting to section 1.
====

.Disambiguating a name shared by a tool and a GED command
====
[source,bash]
----
brlman -S n search
----
The name *search* exists both as a standalone tool (section 1) and as a GED
command (section n). The *-S n* selection shows the GED command page instead
of the default tool page. The equivalent positional form is
`brlman search n`.
====

.Browsing documentation graphically
====
[source,bash]
----
brlman -g
----
Forces the graphical viewer and, because no page was named, opens a browsable
index of all installed programs, libraries, conventions, and GED commands.
====

.Finding a command by a keyword
====
[source,bash]
----
brlman -k matrix transformation
----
Searches manual-page names, synopses, and descriptions for both words. Use
`-K` to search full page text instead.
====
