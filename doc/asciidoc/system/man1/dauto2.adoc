== NAME
dauto2 - Compute the center-clipped autocorrelation function of doubles

== SYNOPSIS

[source]
----
dauto2 [window_size] < input.doubles > output.doubles
----


[[description]]
== DESCRIPTION

*dauto2* reads a stream of double-precision floating-point numbers from standard input, computes an estimate of the autocorrelation function using center-clipping, and writes the resulting double-precision values to standard output. 

The input data is processed in chunks of size _window_size_ (default 512). To reduce the effect of formants in pitch extraction, *dauto2* applies center-clipping to the signal before computing the autocorrelation. The clipping threshold is dynamically determined based on 60% of the maximum absolute peaks found in the first third and the last third of the processing window. The data is then quantized to -1, 0, or 1 before the autocorrelation lags are calculated.

[[options]]
== OPTIONS

*window_size*::
The number of samples per processing window. Defaults to 512 if not specified.

[[examples]]
== EXAMPLES

.Basic *dauto2* Usage
[example]
====
**dauto2 1024 < signal.raw > autocor_clipped.raw**

Computes the center-clipped autocorrelation of the double-precision values in _signal.raw_ using a window size of 1024 samples, and writes the output to _autocor_clipped.raw_.
