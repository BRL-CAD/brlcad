== NAME
window - create a BRL-CAD geometry database of rounded windows

== SYNOPSIS

[source]
----
window [-h] [-f_name.g_] [-n_count_] [-H_height_] [-w_width_] [-d_depth_] [-r_radius_]
----


[[description]]
== DESCRIPTION

*window* creates a BRL-CAD geometry database (a *.g* file) populated with one or
more identical window models, each having rounded edges and corners. Every window
is built from two arb8 solids and four cylinders; the front face is centered at the
origin (0, 0, 0) and the window extends in the negative X direction by its depth.
Up to 256 windows may be created in a single run. *window* uses the *libwdb*
library to write the database and can be used as a starting point for further
editing in *mged*(1).

If invoked with no arguments, *window* prints a usage message and then prompts
interactively for the database name, window count, dimensions, and corner radius.
When any argument is supplied, all parameters are read from the command line and no
prompting occurs. Note that an option and its value are written together with no
intervening space (for example, *-n3*, not *-n 3*).

[[options]]
== OPTIONS

*-h*::
  Print a brief usage and explanation message, then exit. (*-?* behaves the same.)

*-f* _name.g_::
  Name of the output BRL-CAD database to create. Defaults to *window.g* in
  interactive mode if left blank.

*-n* _count_::
  Number of windows to create. Values greater than 256 are clamped to 256.

*-H* _height_::
  Height of each window, in millimeters.

*-w* _width_::
  Width of each window, in millimeters.

*-d* _depth_::
  Depth of each window, in millimeters.

*-r* _radius_::
  Radius of the rounded corners, in millimeters.

[[examples]]
== EXAMPLES

.Generate a set of windows and raytrace them
====
[source,bash]
----
window -fwindows.g -n3 -H200 -w300 -d20 -r30
rt -o windows.png -s512 windows.g wina winb winc
----
Creates a database named *windows.g* containing three identical windows, each
200 mm high, 300 mm wide, 20 mm deep, with 30 mm rounded corners, then renders the
three resulting groups (*wina*, *winb*, *winc*) to an image with *rt*(1).
====
