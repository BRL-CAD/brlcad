== NAME
fbgrid - draw a calibration grid on a frame buffer

== SYNOPSIS

[source]
----
fbgrid [-c] [-b | -d | -o] [-F framebuffer] [-s squaresize] [-w width] [-n height]
----


[[description]]
== DESCRIPTION

*fbgrid* draws a calibration grid pattern on a frame buffer. Three
intensity levels of grid lines are drawn at different spacings, making
it easy to count pixels and verify display geometry. The dark lines are
drawn at intensity 64, the grey lines at 128, and the white lines at 255.

The default grid is decimal (*-d*): dark lines every 10 pixels, grey
lines every 50 pixels, and white lines every 100 pixels.

The frame buffer is selected with *-F* or, if that option is absent, by
the *FB_FILE* environment variable; otherwise the system default device
is used (see *brlcad*(1)).

[[options]]
== OPTIONS

*-c*::
  Clear the frame buffer to black before drawing the grid.

*-b*::
  Use binary spacing: dark lines every 8 pixels, grey lines every 64
  pixels, and white lines every 128 pixels.

*-d*::
  Use decimal spacing (the default): dark lines every 10 pixels, grey
  lines every 50 pixels, and white lines every 100 pixels.

*-o*::
  Draw the old-style grid: a dotted background with white diagonals from
  corner to corner and red lines through the center.

*-F* _framebuffer_::
  Specify the frame buffer device to draw on.

*-s* _squaresize_::
  Set both the width and height of the frame buffer to _squaresize_
  pixels. *-S* is an equivalent synonym.

*-w* _width_::
  Set the frame buffer width to _width_ pixels. *-W* is an equivalent
  synonym.

*-n* _height_::
  Set the frame buffer height to _height_ scanlines. *-N* is an
  equivalent synonym.

*-h*, *-?*::
  Print usage information and exit.


[[examples]]
== EXAMPLES

.Verify the geometry of a rendered image
====
[source,bash]
----
pix-fb -F /tmp/fb moss.pix
fbgrid -F /tmp/fb -d
----
Display a rendered image on a memory frame buffer and overlay a decimal
grid. The grid lines at every 10, 50, and 100 pixels make it easy to
measure feature positions and sizes directly off the screen.
====

.Calibrate a 1024x1024 display with a binary grid
====
[source,bash]
----
fbgrid -c -b -s 1024
----
Clear the frame buffer and draw a binary grid (lines every 8, 64, and
128 pixels) sized to 1024x1024, useful for checking power-of-two display
alignment and pixel addressing.
====
