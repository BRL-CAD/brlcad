== NAME
alias-pix - convert an image from Alias(tm) pix format to BRL PIX(5) format

== SYNOPSIS

[source]
----
alias-pix [-v] ALIASpixfile BRLpixfile
----


[[description]]
== DESCRIPTION

*alias-pix* converts the _Alias pix_ format file on the standard input to an _BRL PIX(5)_ format file on the standard output.


[[options]]
== OPTIONS


*-v*::
print a text summary of the Alias pix file header on standard error output.
