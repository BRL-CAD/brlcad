== NAME

iges-g - convert an IGES file to a BRL-CAD database

== SYNOPSIS

[source]
----
iges-g [-3] [-d] [-n] [-t] [-p] [-N _solid_name_] [-x _rt_debug_flag_] [-X _nmg_debug_flag_] -o _database.g_ _file.iges_
----


[[description]]
== DESCRIPTION

*iges-g* reads an IGES (Initial Graphics Exchange Specification) file and
writes the converted geometry to a BRL-CAD _database.g_ file. The input is
expected to conform to IGES version 5.1.

With none of the *-3*, *-d*, *-n*, or *-t* options, the default action is to
convert only IGES solid model entities (CSG primitives and planar-face BREP
objects) to their BRL-CAD equivalents. Most CSG conversions are exact; solids
of revolution are approximated by a series of BRL-CAD cones and solids of
linear extrusion are approximated by polysolids. Planar-faced BREP objects are
written as BOT solids by default (or as NMG solids with *-p*).

The *-3*/*-d*, *-n*, and *-t* options are mutually exclusive and select
alternate conversion modes (drawings, B-spline surfaces, or trimmed surfaces,
respectively). The *-o* output option is required. When the chosen mode cannot
represent something present in the file, *iges-g* prints suggestions for the
options that would convert the remaining entities. To export geometry in the
other direction, see *g-iges*(1).

[[options]]
== OPTIONS

*-3*::
  Convert IGES drawings to BRL-CAD NMG objects without the final projection
  into 2D, preserving 3D drawings produced from a 3D model. Solid model
  entities are ignored.

*-d*::
  Convert IGES drawings to BRL-CAD NMG objects (projected to 2D). Solid model
  entities are ignored.

*-n*::
  Combine all rational B-spline surfaces in the IGES file into a single
  BRL-CAD spline solid.

*-t*::
  Convert all trimmed surfaces in the IGES file into a single BRL-CAD NMG
  trimmed-NURBS solid.

*-p*::
  Write BREP objects as NMG solids instead of the default BOT solids.

*-o* _database.g_::
  Specify the output BRL-CAD database file. This option is required.

*-N* _solid_name_::
  Name the single solid created by the *-n* or *-t* option. If not given, the
  default name "all" (or a unique variant) is used. Ignored for all other
  modes.

*-x* _rt_debug_flag_::
  Set the librt debug flag, given as a hexadecimal value. See _raytrace.h_ for
  the bit definitions.

*-X* _nmg_debug_flag_::
  Set the NMG routine debug flag, given as a hexadecimal value. See _nmg.h_
  for the bit definitions.

[[examples]]
== EXAMPLES

.Convert IGES solid geometry and inspect the result
====
[source,bash]
----
iges-g -o widget.g widget.iges
mged widget.g 'tops'
----
Convert the CSG and planar-face BREP entities of _widget.iges_ into
_widget.g_, then list the resulting top-level objects with *mged*(1) to verify
the conversion.
====

.Convert trimmed surfaces into a single named solid
====
[source,bash]
----
iges-g -t -N hull -o boat.g boat.iges
----
Collapse all trimmed surfaces in _boat.iges_ into one BRL-CAD NMG solid named
"hull" inside _boat.g_, useful for surface models that have no CSG geometry.
====
