== NAME

nastran-g - convert a NASTRAN finite element model to a BRL-CAD database

== SYNOPSIS

[source]
----
nastran-g [-n] [-m] [-t _tolerance_] [-x _lvl_] [-X _lvl_] [-i _nastran_file_] -o _database.g_
----


[[description]]
== DESCRIPTION

*nastran-g* reads a NASTRAN bulk data deck and converts the geometry it
describes into a BRL-CAD _database.g_ file. Input is read from the file
named with *-i*, or from standard input if *-i* is not given. Conversion
begins at the *BEGIN BULK* card; everything before it is ignored.

Only a subset of NASTRAN element types is handled: *CROD* and *CBAR*
elements become right circular cylinders (RCC) whose radius is derived
from the referenced *PBAR* cross-sectional area, while *CTRIA3* and
*CQUAD4* shell elements are tessellated into surfaces. Grid points
(*GRID*), coordinate systems (*CORD1*/*CORD2*), and properties (*PBAR*,
*PSHELL*, *BAROR*) are interpreted as needed. All geometry is transformed
into the default Cartesian coordinate system. The resulting primitives
are grouped into combinations (_pbars_, _shells_, and a top-level _all_).

By default the shell elements are written as *bot*(5) (BOT) solids and the
model is assumed to be in inches.


[[options]]
== OPTIONS

*-n*::
  Write surfaces created from *CQUAD4* and *CTRIA3* elements as NMG solids
  rather than as the default BOT solids.

*-m*::
  Treat the NASTRAN input as millimeters rather than the default inches.

*-t* _tolerance_::
  Set the distance tolerance, in millimeters: two vertices closer than
  this are treated as the same vertex. Defaults to the BRL-CAD distance
  tolerance.

*-x* _lvl_::
  Set the librt debug flags to the given hexadecimal value.

*-X* _lvl_::
  Set the NMG debug flags to the given hexadecimal value.

*-i* _nastran_file_::
  Read the NASTRAN bulk data deck from _nastran_file_ instead of standard
  input.

*-o* _database.g_::
  Write the converted geometry to _database.g_. Defaults to _nastran.g_.


[[examples]]
== EXAMPLES

.Convert a metric model and inspect the result
====
[source,bash]
----
nastran-g -m -i wing.nas -o wing.g
mged wing.g 'tops'
----
Converts the millimeter NASTRAN deck _wing.nas_ to _wing.g_, then lists
the top-level objects with *mged*(1) so the _all_ combination can be
drawn and examined.
====

.Convert from a pipeline using NMG shells and a tightened tolerance
====
[source,bash]
----
cat model.bdf | nastran-g -n -t 0.005 -o model.g
----
Reads the bulk data deck from standard input, writes shell elements as
NMG solids, and treats vertices within 0.005 mm of each other as
coincident.
====
