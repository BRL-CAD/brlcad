== NAME
3dm-g - convert a Rhino3D (.3dm) model to a BRL-CAD database

== SYNOPSIS

[source]
----
3dm-g [-r] [-v] [-h] [-o output_file.g] input_file.3dm [output_file.g]
----


[[description]]
== DESCRIPTION

*3dm-g* converts the Rhino3D file _input_file.3dm_ into a BRL-CAD geometry
database _output_file.g_. The output file may be named either with the *-o*
option or as the final positional argument.

Because BRL-CAD uses the OpenNURBS library both to read .3dm files and as the
foundation of its own NURBS support, the conversion is typically direct and
reliable. The translation is performed through the *libgcv* geometry conversion
library, so the same import behavior is available from other BRL-CAD tools such
as *mged*(1) and *gcv*.


[[options]]
== OPTIONS

*-r*::
  Randomize the colors assigned to imported objects, which can make distinct
  components easier to tell apart visually.

*-v*, *--verbosity*::
  Increase the verbosity of conversion progress reporting. May be repeated to
  raise the verbosity level further.

*-o*, *--output* _output_file.g_::
  Set the output BRL-CAD database filename. Required if the output file is not
  given as the final positional argument.

*-h*, *--help*, *-?*::
  Print a usage message and exit.


[[examples]]
== EXAMPLES

.Basic conversion
====
[source,bash]
----
3dm-g model.3dm model.g
----
Converts the Rhino3D file _model.3dm_ into a BRL-CAD database named _model.g_.
====

.Verbose conversion with randomized colors and a follow-on render
====
[source,bash]
----
3dm-g -r -v -o nist.g NIST_MBE_PMI_7-10.3dm
rt -o nist.png nist.g NIST_MBE_PMI_7-10
----
Imports the NIST MBE PMI sample model into _nist.g_ with randomized object
colors and detailed progress output, then uses *rt*(1) to ray trace the
resulting top-level assembly to a PNG image for a quick visual check of the
conversion.
====


[[diagnostics]]
== DIAGNOSTICS

OpenNURBS diagnostic messages may be emitted while the .3dm file is read.
