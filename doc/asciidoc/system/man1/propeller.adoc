== NAME
propeller - generates BRL-CAD marine propeller geometry with procedural blades and CSG hubs

== SYNOPSIS

[source]
----
propeller [options]
propeller --mode brep -o propeller_brep.g
propeller --demo-file propeller_demo.g
----

[[description]]
== DESCRIPTION

*propeller* creates a parametric marine propeller model in a BRL-CAD `.g`
database.  The generated model consists of a CSG hub assembly unioned with a
cyclic array of blade solids.  Blade output may be either closed Bag of
Triangles (BoT) solids or smooth BREP blade solids.  The hub remains CSG in
both modes so the outer casing, shaft taper, keyway, and fairing cap remain
available as distinct editable primitives.

The top-level object named by *--name* is a combination containing the generated
region _name_`.r`.  The region contains _name_`_hub_hollow.c` unioned with one
solid per blade.  In BoT mode blade objects are named
_name_`_blade_N.s`; in BREP mode they are named _name_`_blade_N.brep`.

NOTE: the generated models are intended for procedural geometry, rendering,
inspection, and BRL-CAD workflow testing.  They are not validated for marine
engineering design, fabrication, propulsion performance prediction, or
classification-society use.

[[methodology]]
== CONVENTIONS AND METHODOLOGY

The blade generator follows the conventions of Wageningen B-Series style
propeller descriptions: radial stations are evaluated as fractions of
propeller radius, chord length is computed from tabulated spanwise coefficients
as a function of expanded area ratio and blade count, and section ordinates use
empirical back/face factors interpolated over chord fraction and radius.  The
implementation uses those tabulated values as a procedural shape basis rather
than as a certified hydrodynamic design tool.

The coordinate system is centered on the propulsion shaft.  The Z axis is the
shaft line, positive Z is aft, and X/Y form radial planes.  Blade pitch is
applied with a helical transform using the requested tip pitch-diameter ratio.
Rake translates section points along Z, and skew sweeps the blade angularly
using a parabolic radial distribution.

Blade roots are tied to *--hub-diameter-ratio* using `r/R =
hub_diameter_ratio`.  *--root-overlap* intentionally places the innermost blade
station slightly inside that radius to avoid a visible gap where the independent
blade solids meet the hub assembly.  The blade edges are given a small finite
thickness to avoid coincident upper/lower edge geometry in shaded inspection.
The blade tip is closed to an apex rather than by emitting a zero-chord full
section.

The hub generator follows common propeller-hub modeling conventions: an outer
casing is hollowed by a tapered shaft bore, a keyway is subtracted along the
positive Y generator line, and an optional aft fairing cap can be added.  The
supported taper standards are `iso-1-10` and `sae-1-16`; these set the conical
bore half-angle from the corresponding taper ratio.  The keyway subtraction is
a tilted ARB8 slot whose ceiling tracks the shaft taper.  Hub casing profiles
are `cylindrical`, `truncated-cone`, and `convex-barrel`; barrel casings are
written as BoT outer shells, while cylindrical and truncated-cone casings are
written as CSG truncated cone primitives.

In `bot` mode each blade is a closed `RT_BOT_SOLID` written with explicit
surface normals so shaded display can use smoothed side-surface normals while
keeping root and tip closures flat.  In `brep` mode blade side surfaces are
smooth cubic Hermite NURBS patches sharing BREP edges.  Root and tip closures
are trimmed BREP faces, and the generated BREP is checked for OpenNURBS
validity and closed solid topology before it is written.

[[options]]
== OPTIONS

*-h*, *--help*::
Print option help and exit.

*-o* _file.g_, *--output* _file.g_::
Write a single generated propeller to _file.g_.  The default is
`propeller.g`.

*-n* _name_, *--name* _name_::
Use _name_ as the base object name.  The default is `propeller`.

*--demo-file* _file.g_::
Write a sampler database to _file.g_ and exit.  The sampler arranges varied
propeller/hub cases on a regular grid and creates a top-level `all`
combination containing all sample regions.  The built-in sampler covers BoT
and BREP blade output, blade counts from 2 through 7, low and high expanded
area ratios, pitch variation, skew and rake, cylindrical/truncated-cone/barrel
hubs, ISO and SAE shaft tapers, long and short hubs, and fairing/no-fairing
cases.

*--mode* _bot|brep_::
Select blade output geometry.  The default is `bot`.  Hub geometry remains CSG
in both modes.

*-D* _length_, *--diameter* _length_::
Set total propeller diameter.  The default is 1000.

*-Z* _count_, *--blades* _count_::
Set blade count.  The valid range is 2 through 7 and the default is 4.

*--ae-ao* _ratio_::
Set expanded blade area ratio.  The valid range is 0.30 through 1.20 and the
default is 0.65.

*--pd-tip* _ratio_::
Set pitch/diameter ratio at the tip.  The default is 1.0.

*--rake* _degrees_::
Set rake angle in degrees.  Positive values move larger-radius stations aft in
positive Z.  The default is 0.

*--skew* _degrees_::
Set maximum angular skew sweep in degrees.  The default is 0.

*--hub-diameter-ratio* _ratio_::
Set hub diameter divided by propeller diameter.  This also defines the nominal
blade root radius fraction.  The default is 0.20.

*--hub-length-ratio* _ratio_::
Set hub length divided by propeller diameter.  The default is 0.22.

*--hub-profile* _cylindrical|truncated-cone|convex-barrel_::
Select the outer hub casing profile.  The default is `truncated-cone`.

*--hub-forward-diameter* _length_::
Set forward casing diameter.  If omitted, this is derived from
*--hub-diameter-ratio*.

*--hub-aft-diameter* _length_::
Set aft casing diameter.  If omitted, this is derived from the selected hub
profile.

*--hub-barrel-swell* _ratio_::
Set the convex barrel swell ratio.  The default is 1.08.  This option is used
by `convex-barrel` hubs.

*--shaft-standard* _iso-1-10|sae-1-16_::
Select the internal shaft taper convention.  The default is `iso-1-10`.

*--shaft-diameter-small* _length_::
Set the small-end shaft bore diameter.  If omitted, a conservative value is
derived from hub diameter.

*--keyway-width* _length_::
Set keyway slot width.  If omitted, a value is derived from shaft diameter.

*--keyway-depth* _length_::
Set keyway depth into the hub.  If omitted, a value is derived from shaft
diameter.

*--fairing-cap-length* _length_::
Set aft fairing cap length.  If omitted, a value is derived from propeller
diameter.

*--no-fairing-cap*::
Do not add the aft fairing cap.

*--hub-slices* _count_::
Set longitudinal slice count for generated barrel hub BoT shells.  The default
is 30.

*--hub-segments* _count_::
Set angular segment count for generated barrel hub BoT shells.  The default is
72.

*--root-overlap* _ratio_::
Set how far the innermost blade station extends inside the nominal hub radius
in `r/R` units.  The default is 0.01.

*--edge-thickness-ratio* _ratio_::
Set finite leading/trailing blade edge thickness divided by propeller diameter.
The default is 0.00075.

*--radial-stations* _count_::
Set blade radial section count, excluding the tip apex.  The minimum is 4 and
the default is 20.

*--chord-points* _count_::
Set chordwise samples per upper/lower blade grid.  The minimum is 5 and the
default is 50.

*--x-offset* _distance_, *--y-offset* _distance_, *--z-offset* _distance_::
Translate the generated model in database coordinates.

*-f*, *--force*::
Overwrite an existing output file.

*-j*, *--append*::
Append objects to an existing output database.  Object name collisions are
rejected.

[[examples]]
== EXAMPLES

Create the default propeller as BoT blades with a CSG hub:

[source,sh]
----
propeller -o propeller.g
----

Create smooth BREP blades while keeping the hub as CSG:

[source,sh]
----
propeller --mode brep -o propeller_brep.g -f
----

Generate a highly skewed five-blade propeller with a truncated-cone hub:

[source,sh]
----
propeller -o skewed5.g -f -Z 5 --ae-ao 0.95 --pd-tip 1.18 \
  --skew 28 --rake 4 --hub-diameter-ratio 0.22 \
  --hub-profile truncated-cone --shaft-standard sae-1-16
----

Generate a low-pitch three-blade propeller with a convex barrel hub and no
fairing cap:

[source,sh]
----
propeller -o barrel3.g -f -Z 3 --ae-ao 0.55 --pd-tip 0.75 \
  --hub-profile convex-barrel --hub-barrel-swell 1.15 \
  --no-fairing-cap --hub-segments 48 --hub-slices 18
----

Generate a dense BoT blade mesh for closer visual inspection:

[source,sh]
----
propeller -o dense_bot.g -f --radial-stations 32 --chord-points 80
----

Generate visibly blunter blade edges for shaded inspection:

[source,sh]
----
propeller -o blunt_edges.g -f --edge-thickness-ratio 0.0015
----

Generate the built-in inspection grid:

[source,sh]
----
propeller --demo-file propeller_demo.g
----

Append two named propellers to the same database at different locations:

[source,sh]
----
propeller -o comparison.g -f -n low_pitch --pd-tip 0.75 --x-offset 0
propeller -o comparison.g -j -n high_pitch --pd-tip 1.35 --x-offset 1500
----
