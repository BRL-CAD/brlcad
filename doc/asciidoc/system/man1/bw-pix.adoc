== NAME
bw-pix - convert a black and white image to color format

== SYNOPSIS

[source]
----
bw-pix [ [-s squaresize] [-w width] [-n height]] [-o out.pix] [ [<]in.bw] > [out.pix]
----


[[description]]
== DESCRIPTION

*bw-pix* converts a black and white *bw*(5) format file to a color *pix*(5) format file. Each input value is replicated three times, thus the resulting output file has the same value in all three color planes. An input file name can be given, else input comes from stdin. Output will go to stdout unless an optional output file name is given.
