== NAME
dpix-pix - convert a double-precision dpix file to a pix file

== SYNOPSIS

[source]
----
dpix-pix file.dpix > file.pix
----

[[description]]
== DESCRIPTION

*dpix-pix* converts a double-precision image (dpix) file to a *pix*(5)
format file.  It scans the entire input file once to find the minimum
and maximum sample values, then reads the file a second time and
linearly maps each value from that [min, max] range onto the 8-bit
range of 0 to 255, writing the resulting bytes to standard output.  The
discovered minimum and maximum are reported on standard error.

Because the input must be read twice, the data is taken from a named
file rather than from standard input.  The output is binary and is
refused if directed to a terminal.

[[examples]]
== EXAMPLES

.Convert a dpix file and view the result
====
[source,bash]
----
dpix-pix image.dpix > image.pix
pix-fb -w 512 -n 512 image.pix
----
Converts the double-precision image `image.dpix` into a *pix*(5) file,
auto-scaling its values into the 0-255 range, then displays the result
on a framebuffer with *pix-fb*(1).  The reported min/max on standard
error indicate the exposure range used for the conversion.
====
