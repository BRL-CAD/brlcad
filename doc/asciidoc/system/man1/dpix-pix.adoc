== NAME
dpix-pix - convert a double precision dpix file to a pix file

== SYNOPSIS

[source]
----
dpix-pix file.dpix > file.pix
----


[[description]]
== DESCRIPTION

Outputs a dpix file to pix (see *pix*(5)) format. Standard input is read if a file name is not supplied.
