== NAME
pixcolors - Count/list color values used in a PIX(5) format image

== SYNOPSIS

[source]
----
pixcolors [-v] < pixfile
----


[[description]]
== DESCRIPTION

*pixcolors* reports on the standard output the number of distinct colors used in the _pix(5)_ format file on the standard input.


[[options]]
== OPTIONS


*-v*::
Causes a sorted list of the unique colors used to be printed on standard output in addition to the count report.
