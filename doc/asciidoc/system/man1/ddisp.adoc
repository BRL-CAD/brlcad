== NAME
ddisp - plot a stream of doubles on a framebuffer

== SYNOPSIS

[source]
----
ddisp [-v] [-b] [-p] [-c] [-H] < input.doubles
----


[[description]]
== DESCRIPTION

*ddisp* reads a continuous stream of host double-precision floating-point
numbers from standard input and plots them onto a 512x512 BRL-CAD
framebuffer. The data is processed in records of _BU_PAGE_SIZE_ values at a
time, with each record clamped to the 512-pixel framebuffer width.

In the default mode, each record is drawn as one horizontal grayscale line.
Every value is scaled (magnitude 0.0 to 1.0 mapped to intensity 0-255) and
written as a pixel; successive records march down the screen, wrapping back to
the top when the bottom is reached. The *-v* and *-b* options instead plot
each record as a green waveform centered vertically, where a value of +/- 1.0
maps to +/- 128 pixels from the center row.

Input is typically produced by signal-processing tools such as *dmod*(1),
*dfft*(1), or *dconv*(1). If standard input is a terminal, *ddisp* prints its
usage and exits.

[[options]]
== OPTIONS

*-v*::
  Vertical (point) plot mode. Each value is plotted as a single green pixel
  whose height above or below the center row is proportional to its value.
  This option also disables pausing and clearing.

*-b*::
  Bar graph mode. Like *-v*, but draws a solid green vertical bar from the
  center row out to each value, clamped to the +/- 128 pixel range.

*-p*::
  Pause for 3 seconds after drawing each record. Useful for animating
  successive records frame by frame.

*-c*::
  Clear the framebuffer before drawing each record. Most useful with *-v* or
  *-b* to produce a clean animation of each frame.

*-H*::
  Print the usage message and exit. The *-h* and *-?* flags behave the same
  way.

[[examples]]
== EXAMPLES

.Animate a waveform as a bar graph
====
[source,bash]
----
dmod -s2.0 < signal.d | ddisp -b -c -p
----
Scales a stream of doubles with *dmod*(1), then feeds the result to *ddisp*,
which clears the framebuffer and draws each record as a green bar graph,
pausing three seconds between frames to animate the signal.
====

.Display a scrolling intensity map
====
[source,bash]
----
ddisp < spectrogram.d
----
Reads records of doubles from _spectrogram.d_ and draws each record as a
horizontal grayscale line, scrolling down the framebuffer to build up a
spectrogram-style display.
====
