== NAME
plot3-asc - convert a Plot3 stream to ASCII

== SYNOPSIS

[source]
----
plot3-asc [-v] [unix_plot]
----


[[description]]
== DESCRIPTION

Reads a stream in *plot3*(5) format and converts it to an ASCII representation. Standard input is read if a file name is not supplied. If the *-v* option is given, the single-character command names are replaced with mnemonic strings and a summary of plot command usage is written to standard error.
