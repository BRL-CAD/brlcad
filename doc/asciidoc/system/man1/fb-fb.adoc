== NAME
fb-fb - copy a framebuffer image to another framebuffer

== SYNOPSIS

[source]
----
fb-fb [-v] [-F output_framebuffer] [-h] [-?] input_framebuffer [output_framebuffer]
----

[[description]]
== DESCRIPTION

*fb-fb* copies the image from _input_framebuffer_ to the output
framebuffer. The output device may be specified with *-F* or as the
second positional argument.

[[options]]
== OPTIONS

*-v*::
Enable verbose mode.

*-F* _output_framebuffer_::
Explicitly specify the output framebuffer device.

*-h, -?*::
Print a usage message and exit.

[[examples]]
== EXAMPLES

.Copy the default framebuffer to a file-based framebuffer
====
[source,bash]
----
$ fb-fb /dev/Xl /dev/debug
----
====
