== NAME
btclsh - Tcl-based interactive shell for BRL-CAD

== SYNOPSIS

[source]
----
btclsh [script]...
----


== DESCRIPTION

*btclsh* is a Tcl-based interactive shell for BRL-CAD. It provides command-line editing, command history, and scripting capabilities. It is particularly suited for text-based interactive scripting and automation in BRL-CAD environments.

Unlike *bwish*, *btclsh* does not require a graphical interface and can be used in environments without a display server.


== OPTIONS


_script_::
Executes the provided Tcl script file and exits.
