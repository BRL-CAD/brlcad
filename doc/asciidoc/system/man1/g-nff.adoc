== NAME

g-nff - BRL-CAD to Neutral File Format (NFF) translator

== SYNOPSIS

[source]
----
g-nff [-v] [-i] [-x _rt_debug_] [-X _nmg_debug_] [-e _error_file_] [-D _dist_tol_] [-a _abs_tol_] [-r _rel_tol_] [-n _norm_tol_] -o _output.nff_ _database.g_ _object_ ...
----

[[description]]
== DESCRIPTION

*g-nff* converts the specified geometry _object(s)_ from a BRL-CAD _database.g_
file to the Neutral File Format (NFF). Each region is evaluated with the NMG
boolean routines, triangulated, and written as a set of NFF polygons tagged
with the region's color. *g-nff* also computes a bounding box for the geometry
and emits a default 45-degree view, light, and 512x512 resolution into the
output so the result can be rendered directly.

[[options]]
== OPTIONS

*-o* _output.nff_::
  Specify the output NFF file name. This option is required.

*-i*::
  Treat the output as being in inches. The default is millimeters.

*-a* _abs_tol_::
  Specify the absolute tessellation tolerance in millimeters. Setting this
  disables the relative tolerance.

*-r* _rel_tol_::
  Specify the relative tessellation tolerance as a fraction of the bounding box
  diagonal. The default is 0.01.

*-n* _norm_tol_::
  Specify the surface normal tessellation tolerance in degrees. Setting this
  disables the relative tolerance.

*-D* _dist_tol_::
  Specify the calculational distance tolerance in millimeters used when
  deciding whether two points are coincident.

*-e* _error_file_::
  Redirect error messages to _error_file_ instead of standard error.

*-v*::
  Enable verbose progress reporting.

*-x* _rt_debug_::
  Set the librt debug flags (given as a hexadecimal value).

*-X* _nmg_debug_::
  Set the libnmg debug flags (given as a hexadecimal value).

*-h*, *-?*::
  Print the usage message and exit.

[[examples]]
== EXAMPLES

.Convert selected regions to a renderable NFF file
====
[source,bash]
----
g-nff -a 0.5 -o engine.nff engine.g block piston
----
Tessellates the *block* and *piston* regions from *engine.g* using a 0.5 mm
absolute tolerance and writes the triangulated geometry, together with a
default view and light, to *engine.nff*, ready to be passed to an NFF-capable
renderer.
====
