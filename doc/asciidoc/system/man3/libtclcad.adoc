== NAME
libtclcad - Tcl/Tk integration library for BRL-CAD

== SYNOPSIS

[source,c]
----
#include <tclcad.h>
----

Link with *-ltclcad*.

The public interface is organized across the headers included by
_tclcad.h_, namely _tclcad/defines.h_, _tclcad/setup.h_,
_tclcad/draw.h_, and _tclcad/misc.h_.

== DESCRIPTION

LIBTCLCAD contains convenience routines for preparing and initializing
Tcl (and Tk) for use with BRL-CAD. It provides the setup logic that
registers BRL-CAD functionality with a Tcl interpreter and supplies the
drawing and utility helpers used by BRL-CAD's Tcl/Tk-based applications,
such as *mged*(1) and *archer*(1).

The library ties the geometry editing capabilities of libged and the
display managers of libdm to a scripting environment, exposing them so
that geometry can be created, queried, edited, and rendered from Tcl
scripts and interactive command shells.

The complete API, including data structures and function signatures, is
documented in the headers noted above and in the Doxygen-generated
reference for the LIBTCLCAD module.
