== NAME
libpc - parametrics and constraints library for BRL-CAD

== SYNOPSIS

[source,c]
----
#include <pc.h>
----

Link with *-lpc*.

== DESCRIPTION

LIBPC provides the data structures and evaluation interface for
parametrics and constraints in BRL-CAD. Its scope is to serve as a
wrapper over a geometric constraint evaluation system, defining the
representations for parameters, constraints, and constraint sets used to
describe relationships between geometric quantities.

The library is intended to underpin parametric constraint solving and
joint articulation, with the expectation that one or more third-party
solvers may be incorporated to evaluate the constraint networks. The
longer-term intent is for librt to rely on this library for the
associated data types and evaluation.

The complete API, including data structures and function signatures, is
documented in _pc.h_ and its included files, and in the
Doxygen-generated reference for the LIBPC module.
