== NAME
libqtcad - Qt widget library for BRL-CAD

== SYNOPSIS

[source,cpp]
----
#include <qtcad/QgView.h>
----

Link with *-lqtcad*.

The public interface is provided by the individual _qtcad/Qg*.h_
headers, such as _qtcad/QgView.h_, _qtcad/QgModel.h_,
_qtcad/QgTreeView.h_, _qtcad/QgConsole.h_, and _qtcad/QgSelectFilter.h_,
together with _qtcad/defines.h_.

== DESCRIPTION

LIBQTCAD is a C++ library of Qt widgets and supporting classes for
building BRL-CAD graphical user interfaces. It provides the reusable
components used by the Qt-based front ends, including view and quad-view
widgets, a geometry tree view and selection model, an attribute editor,
an interactive command console, geometry import dialogs, and assorted
tool palettes and helper widgets.

The widgets bridge Qt's model/view and event system to BRL-CAD's
geometry libraries, wrapping libged commands and libdm display managers
so that models can be displayed, navigated, selected, and edited within
a Qt application.

Because it depends on Qt, LIBQTCAD is built only when Qt support is
enabled in the BRL-CAD configuration. The complete API is documented in
the headers noted above and in the Doxygen-generated reference for the
LIBQTCAD module.
