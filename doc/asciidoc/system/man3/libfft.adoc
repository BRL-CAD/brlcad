== NAME
libfft - Fast Fourier Transform library for BRL-CAD

== SYNOPSIS

[source,c]
----
#include <fft.h>
----

Link with *-lfft*.

== DESCRIPTION

LIBFFT is a signal-processing library for performing Fast Fourier
Transforms (FFTs) and inverse FFTs efficiently. It provides forward and
inverse one- and two-dimensional transforms, including routines
specialized for particular fixed transform sizes, operating on real and
complex data.

The library defines a simple _COMPLEX_ type consisting of real and
imaginary double-precision components, and intentionally avoids a
dependency on _vmath.h_ so that it may be used as a standalone signal
processing component.

The complete API, including data structures and function signatures, is
documented in _fft.h_ and in the Doxygen-generated reference for the
LIBFFT module.
