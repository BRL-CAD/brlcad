Tcl/Tk

  • Tcl is an embeddable, full-function interpreted command language
  • Tk is a toolkit, based on Tcl, for building user interfaces, etc.
  • Created by John Ousterhout
    <john.ousterhout@scriptics.com>
  • Available from ftp://ftp.scriptics.com/pub/tcl
