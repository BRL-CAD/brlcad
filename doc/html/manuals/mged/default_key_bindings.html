Default Key Bindings

Key Command Description
a adc toggle adc cursor
e set e_axes ! toggle edit axes
v set v_axes ! toggle view axes
w set w_axes ! toggle world axes
i aip f advance illumination pointer forward
I aip b advance illumination pointer backward
p M 1 0 0 simulate mouse click
3 press 35,25 set view azimuth/elevation to 35/25
4 press 45,45 set view azimuth/elevation to 45/45
f press front set view azimuth/elevation to 0/0
t press top set view azimuth/elevation to -90/90
b press bottom set view azimuth/elevation to -90/-90
l press left set view azimuth/elevation to 90/0
r press right set view azimuth/elevation to 270/0
R press rear set view azimuth/elevation to 180/0
s press sill solid illuminate
o press oill object illuminate
q press reject reject edit
u svb set view reference base
n nirt -b shoot NIRT ray at center of view

<Control-p> set view 0 advance to previous view
