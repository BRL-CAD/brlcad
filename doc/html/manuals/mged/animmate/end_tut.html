Conclusion

Congratulations! You have now learned how to create object, view, and track animation scripts with AnimMate.

Since you are probably interested in obtaining ray-traced images of your animations, Appendix A includes a brief demonstration of how to use the BRL-CAD ray-tracer for animations.

Much of AnimMate's functionality comes from a set of BRL-CAD animation routines; Appendix B contains a list of these routines, with links to the corresponding manual pages. You might want to consult these manual pages for a better understanding of how AnimMate works, or to learn how to some of their features which are not available through the AnimMate interface.

Any comments, questions, or bug reports that you have regarding this tutorial or AnimMate itself are welcome, and should be sent via e-mail to brlcad-users@lists.sourceforge.net.
