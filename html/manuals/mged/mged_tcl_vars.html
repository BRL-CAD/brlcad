MGED Variables

The use of pathName in this document refers to the Tcl/Tk pathname of a display manager window.

local2base
local2base represents the conversion factor that converts local units to base units (i.e. millimeters).


base2local
base2local represents the conversion factor that converts base units (i.e. millimeters) to local units.


rate_tran(pathName,X)
rate_tran(pathName,X) represents the rate of X translation for pathName's view.


rate_tran(pathName,Y)
rate_tran(pathName,Y) represents the rate of Y translation for pathName's view.


rate_tran(pathName,Z)
rate_tran(pathName,Z) represents the rate of Z translation for pathName's view.


rate_model_tran(pathName,X)
rate_model_tran(pathName,X) represents the rate of X translation, in model coordinates, for pathName's view.


rate_model_tran(pathName,Y)
rate_model_tran(pathName,Y) represents the rate of Y translation, in model coordinates, for pathName's view.


rate_model_tran(pathName,Z)
rate_model_tran(pathName,Z) represents the rate of Z translation, in model coordinates, for pathName's view.


rate_rotate(pathName,X)
rate_rotate(pathName,X) represents the rate of X rotation for pathName's view.


rate_rotate(pathName,Y)
rate_rotate(pathName,Y) represents the rate of Y rotation for pathName's view.


rate_rotate(pathName,Z)
rate_rotate(pathName,Z) represents the rate of Z rotation for pathName's view.


rate_model_rotate(pathName,X)
rate_model_rotate(pathName,X) represents the rate of X rotation, in model coordinates, for pathName's view.


rate_model_rotate(pathName,Y)
rate_model_rotate(pathName,Y) represents the rate of Y rotation, in model coordinates, for pathName's view.


rate_model_rotate(pathName,Z)
rate_model_rotate(pathName,Z) represents the rate of Z rotation, in model coordinates, for pathName's view.


rate_scale(pathName)
rate_scale represents the rate of scale for pathName's view.


abs_tran(pathName,X)
abs_tran(pathName,X) represents the absolute distance in X the view has been translated from its home position.


abs_tran(pathName,Y)
abs_tran(pathName,Y) represents the absolute distance in Y the view has been translated from its home position.


abs_tran(pathName,Z)
abs_tran(pathName,Z) represents the absolute distance in Z the view has been translated from its home position.


abs_model_tran(pathName,X)
abs_model_tran(pathName,X) represents the absolute distance in model X the view has been translated from its home position.


abs_model_tran(pathName,Y)
abs_model_tran(pathName,Y) represents the absolute distance in model Y the view has been translated from its home position.


abs_model_tran(pathName,Z)
abs_model_tran(pathName,Z) represents the absolute distance in model Z the view has been translated from its home position.


abs_rotate(pathName,X)
abs_rotate(pathName,X) represents the amount of X rotation that has been applied to the view.


abs_rotate(pathName,Y)
abs_rotate(pathName,Y) represents the amount of Y rotation that has been applied to the view.


abs_rotate(pathName,Z)
abs_rotate(pathName,Z) represents the amount of Z rotation that has been applied to the view.


abs_model_rotate(pathName,X)
abs_model_rotate(pathName,X) represents the amount of X rotation, in model coordinates, that has been applied to the view.


abs_model_rotate(pathName,Y)
abs_model_rotate(pathName,Y) represents the amount of Y rotation, in model coordinates, that has been applied to the view.


abs_model_rotate(pathName,Z)
abs_model_rotate(pathName,Z) represents the amount of Z rotation, in model coordinates, that has been applied to the view.


abs_scale(pathName)
abs_scale(pathName) represents the amount of scaling applied to the view. The normal range is [-1.0,1.0].


xadc(pathName)
xadc(pathName) represents the amount of X translation the angle distance cursor has moved. The range is [-2048,2047].


yadc(pathName)
yadc(pathName) represents the amount of Y translation the angle distance cursor has moved. The range is [-2048,2047].


ang1(pathName)
ang1(pathName) represents the angle distance cursors' first angle. The range is [-180,180].


ang2(pathName)
ang2(pathName) represents the angle distance cursors' second angle. The range is [-180,180].


distadc(pathName)
distadc(pathName) represents the angle distance cursors' tick position. The range is [-2048,2047].


Viewscale(pathName)
Viewscale(pathName) is representative of the view size. More specifically, view size is equal to 2 times Viewscale(pathName).


edit_class
edit_class represents the class of edit that is in effect. For example, 0 signifies none, 1 signifies translation, 2 signifies rotation and 3 signifies scale.


edit_solid_flag
edit_solid_flag specifies the type of edit to perform on the solid that is being edited. edit_solid_flag is linked to MGED's internal variable es_edflag.


edit_object_flag
edit_object_flag specifies the type of edit to perform on the object that is being edited. edit_object_flag is linked to MGED's internal variable edobj.

edit_rate_model_tran(X) represents the rate of X translation, in model coordinates, for the solid/object being edited.


edit_rate_model_tran(Y)
edit_rate_model_tran(Y) represents the rate of Y translation, in model coordinates, for the solid/object being edited.

edit_rate_object_rotate(Y)
edit_rate_object_rotate(Y) represents the rate of Y rotation, in object coordinates, for the solid/object being edited.


edit_rate_object_rotate(Z)
edit_rate_object_rotate(Z) represents the rate of Z rotation, in object coordinates, for the solid/object being edited.


edit_rate_view_rotate(X)
edit_rate_view_rotate(X) represents the rate of X rotation, in view coordinates, for the solid/object being edited.


edit_rate_view_rotate(Y)
edit_rate_view_rotate(Y) represents the rate of Y rotation, in view coordinates, for the solid/object being edited.


edit_rate_view_rotate(Z)
edit_rate_view_rotate(Z) represents the rate of Z rotation, in view coordinates, for the solid/object being edited.


edit_rate_scale
edit_rate_scale represents the rate of scale for the solid/object being edited.


edit_abs_model_tran(X)
edit_abs_model_tran(X) represents the absolute X translation, in model coordinates, for the solid/object being edited.


edit_abs_model_tran(Y)
edit_abs_model_tran(Y) represents the absolute Y translation, in model coordinates, for the solid/object being edited.

edit_abs_object_rotate(X) represents the absolute X rotation, in object coordinates, for the solid/object being edited.

edit_abs_object_rotate(Y) represents the absolute Y rotation, in object coordinates, for the solid/object being edited.

edit_abs_view_rotate(X)

edit_abs_scale

mged_display(state)
mged_display(state) specifies the state of MGED. The state string will be one of the six values listed below.
  • VIEWING
  • SOL PICK
  • SOL EDIT
  • OBJ PICK
  • OBJ PATH
  • OBJ EDIT


mged_display(pathName,fps)
mged_display(pathName,fps) specifies the frames per second that MGED is able to display.


mged_display(pathName,aet)
mged_display(pathName,aet) specifies the current azimuth, elevation and twist for pathName.


mged_display(pathName,ang)
mged_display(pathName,ang) indicates the rates of view rotation.


mged_display(pathName,center)
mged_display(pathName,center) indicates the view center.


mged_display(pathName,size)
mged_display(pathName,size) indicates the view size.


mged_display(pathName,adc)
mged_display(pathName,adc) specifies information about the angle distance cursor.


edit_info
edit_info contains detailed information about the current solid/object being edited.


mged_display(path_lhs)
mged_display(path_lhs) indicates the left hand side path.


mged_display(path_rhs)
mged_display(path_rhs) indicates the right hand side path.


mged_display(units)
mged_display(units) indicates the local units.


mged_display(keypoint)
mged_display(keypoint) indicates the point where edits will take place from.
