LIBDM's Tcl Interface

dm_best_type
dm_best_type returns the best available display manager type.

dm_best_name
dm_best_name returns the best available display manager name.

dm_name2type name
dm_name2type return the display manager type that corresponds to name.

dm_names
dm_names returns a list of available display manager names.

dm_type2name type
dm_type2name returns the display manager name that corresponds to type.

dm_types
dm_types returns a list of available display manager types.
