Keysym List

This section contains a list of the supported keysyms which were taken from the keysymdef.h header file that is provided with the X11 distribution. The keysyms are used with the bindkey command and in the startup file for specifying key bindings. When using the keysym with the bindkey command, the preceding XK_ is not used. The following sections of the header file are contained in the list below:

  • Miscellany
  • Japanese keyboard support
  • Cursor control & motion
  • Misc Functions
  • Keypad Functions
  • Auxilliary Functions
  • Modifiers
  • Latin1
