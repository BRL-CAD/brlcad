XMGED - An X Interface to MGED

This document describes enhancements and changes made to the BRL-CAD MGED solid model editor. While called XMGED, it has been implemented primarily as a new display manager in MGED, and as such all of the old interfaces still operate. Besides offering a more modern user interface look and feel, there are also numerous new features including command history and editing, journaling and script files, multiple views, alternate mouse modes, position (rather than rate) based knobs, etc.
