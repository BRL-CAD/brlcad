Scale H Command

Figure (esc-sh): Cylinder Scale H Vector

mged> Select the ``edit menu'' entry in the solid edit menu
mged> Select the ``scale H'' entry in the TGC menu
mged> p 1
mged>

The magnitude of the vector H is reduced from 2 to 1. See Figure (esc-sh) . The command

mged> p 2
mged>

will restore the original shape.
