Future Work

These are a few of the items planned for the future. It should be noted that all of the XMGED features are still subject to change based on user feedback and future design decisions.

  • Better help facility. We are looking into a hypertext help facility.
  • Provide a choice between using the Motif or Athena widget sets
  • Read startup file that contains specifications for menus, key bindings, text colors, and the default views. These descriptions would be something like that found in a .mwmrc or .twmrc file.
  • Command line editing for mged proper
  • Display view axis
  • Handle resize
