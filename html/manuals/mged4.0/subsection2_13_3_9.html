Move End H Command

Figure (esc-mh): Cylinder Move End of H

mged> Select the ``move end H'' entry in the TGC menu
mged> p 3
mged>

The length of the vector H is changed to the value of p. See Figure (esc-mh) . The command

mged> p 2
mged>

will restore the original shape.
