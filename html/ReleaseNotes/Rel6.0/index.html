Summary

[back]
  • New Macintosh OS X architecture support
  • Improved raytrace performance
  • New database file format and migration tools
  • New ascii database file format
  • Ascii support for extruded bitmap (ebm) and volume (vol) primitives
  • Improved height field (dsp) primitive performance (Bresenham voxel traversal)
  • A new "sketch" primitive which is a 2D sketch consisting of straight lines, circular arcs, and bezier curves
  • A new "extrusion" primitive which allows linear extrusion of closed planar sketches
  • Tesselation support for sketch extrusion (extrude) primitve
  • General bug fixes, security enhancements, and speed improvements
  • Multi-device Geometry Editor (MGED) improvements
    • New "accelerators" to automatically generate more complex primitives from existing ones: rcc-blend, rcc-cap, rcc-tgc, rpp-arch, rpp-cap, rpp-arch, tor-rcc, sph-part
    • Improved menu and interface layout
    • New "Introduction to MGED" user manual
    • 'in' command support for kinematics control (grip), sketch extrusion (extrude), arbitrary regular polyhedron (arbn), and binary object (binunif) primitives
    • 'make' command support for rectangular parallelopipeds (rpp), arbitrary faceted (ars), and arbitrary regular polyhedron (arbn) primitives
    • New commands include: adjust, autoview, attr, attr_rm, copyeval, hide, lt, nmg_collapse, nmg_simplify, pathlist, shells, showmats, summary, unhide, version, xpush, rtedge, E, bot_face_sort
    • Bezier curves are interactive display support
    • New "Build Pattern Tool" for duplicating geometry in a pattern
    • New attribute editor panel
  • Supported architectures include Linux, FreeBSD, SGI IRIX, and Sun
  • Improved Pro-Engineer conversion support (new converters)
    • Oriented and non-oriented triangle support
    • Unlimited name length support
  • Fastgen 4 conversion now supports character names longer than 16 characters
  • Hidden geometry support
  • Geometry databases now support attributes
  • Improved and updated support documentation (man pages, command help, and documentation)
  • Improved raytracer edge detection performance
  • Initial support for multiply represented objects (MROs)
  • Improved Bezier curve support for sketch extrusion (extrude) primitive
  • Binary object support for Height field (dsp) primitive and texture shader
  • New flat and invisible shaders

New Utilities

[back]
  • pixcmp

    Although not technically new (formerly included in source distributions), pixcmp is now installed

  • rtedge

    Raytraces models, detects edges, and writes BRL-CAD color image (.pix) files

  • g_diff

    Compares two geometry database files and reports differences between the two

New Libraries

[back]
  • New general purpose bourne-shell utility library (part of regression suite)

    Includes support for file locking, semaphores, stdout/stderr/file logging, mail, cvs, packaging, and remote host querying facilities

Developer Notes

[back]
  • New and improved regression test suite
  • Compile-time optimizations that significantly increase raytrace performance (at cost of run-time debugging)
  • Detached dependancy on X (and Tk) during source compilation
  • Consolidated cake build options
  • Mged 'db' command facilities
  • Improved ANSI source conformance (still in transition)

Deprecated Features

[back]
  • lgt

    lgt is being replaced by the new 'rtedge' tool

Unsupported Features and Utilities

[back]
  • g4-g5

    This tool has been replaced by the more general 'dbupgrade' command

  • Standard input/output redirection is no longer supported for g2asc, asc2g
